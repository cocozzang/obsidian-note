# 함수의 극대와 극소(2)

## 1. 예제

### 예제 207

함수 $f(x) = 2x^3 - 9x^2 + 12x + 2$의 극댓값을 $M$, 극솟값을 $m$이라 할 때 $Mm$의 값을 구하여라.

> [!summary]- 풀이
> **Step 1: 극점 찾기**
> 
> $$f'(x) = 6x^2 - 18x + 12 = 0$$
> 
> $$6(x^2 - 3x + 2) = 0$$
> 
> $$6(x - 1)(x - 2) = 0$$
> 
> $$x = 1 \quad \text{or} \quad x = 2$$
> 
> **Step 2: 극값 판정**
> 
> **방법 1 (증감표)**:
> 
> $$f'(0) = 12 > 0, \quad f'\left(\frac{3}{2}\right) = 6 \cdot \frac{9}{4} - 27 + 12 < 0, \quad f'(3) = 30 > 0$$
> 
> | $x$ | $\cdots$ | $1$ | $\cdots$ | $2$ | $\cdots$ |
> |-----|----------|-----|----------|-----|----------|
> | $f'(x)$ | $+$ | $0$ | $-$ | $0$ | $+$ |
> | $f(x)$ | $\nearrow$ | 극대 | $\searrow$ | 극소 | $\nearrow$ |
> 
> 따라서 $x = 1$에서 극대, $x = 2$에서 극소
> 
> **방법 2 (2계 도함수)**:
> 
> $$f''(x) = 12x - 18$$
> 
> $$f''(1) = -6 < 0 \quad \Rightarrow \quad x = 1 \text{에서 극대}$$
> 
> $$f''(2) = 6 > 0 \quad \Rightarrow \quad x = 2 \text{에서 극소}$$
> 
> **Step 3: 극값 계산**
> 
> $$M = f(1) = 2 - 9 + 12 + 2 = 7$$
> 
> $$m = f(2) = 16 - 36 + 24 + 2 = 6$$
> 
> $$Mm = 7 \times 6 = 42$$

### 예제 208

함수 $f(x) = x^3 - 3ax^2 + 4a$의 그래프가 $x$축과 접할 때 (극점이 $x$축에 접함) $a$의 값을 구하여라. (단, $a > 0$)

> [!summary]- 풀이
> **Step 1: 극점 찾기**
> 
> $$f'(x) = 3x^2 - 6ax = 3x(x - 2a) = 0$$
> 
> $$x = 0 \quad \text{or} \quad x = 2a$$
> 
> **Step 2: 극값 판정 (2계 도함수)**
> 
> $$f''(x) = 6x - 6a$$
> 
> $$f''(0) = -6a < 0 \quad \Rightarrow \quad x = 0 \text{에서 극대}$$
> 
> $$f''(2a) = 12a - 6a = 6a > 0 \quad \Rightarrow \quad x = 2a \text{에서 극소}$$
> 
> **Step 3: 극값 계산**
> 
> $$f(0) = 4a > 0$$
> 
> 극댓값이 양수이므로 극대점은 $x$축과 접하지 않음
> 
> $$f(2a) = 8a^3 - 12a^3 + 4a = -4a^3 + 4a$$
> 
> **Step 4: 극소점이 $x$축과 접하는 조건**
> 
> $$f(2a) = 0$$
> 
> $$-4a^3 + 4a = 0$$
> 
> $$4a(1 - a^2) = 0$$
> 
> $$a(1 - a)(1 + a) = 0$$
> 
> $$a = 0 \quad \text{or} \quad a = \pm 1$$
> 
> $a > 0$이므로 **$a = 1$**

### 예제 209

두 상수 $a, b$에 대하여 함수 $f(x) = x^3 + ax^2 + 9x + b$가 $x = 1$에서 극댓값 $0$을 가질 때 $ab$의 값을 구하여라.

> [!summary]- 풀이
> **조건 1**: $x = 1$에서 극값 $\Rightarrow$ $f'(1) = 0$
> 
> $$f'(x) = 3x^2 + 2ax + 9$$
> 
> $$f'(1) = 3 + 2a + 9 = 0$$
> 
> $$2a = -12$$
> 
> $$a = -6$$
> 
> **조건 2**: $x = 1$에서 극댓값이 $0$ $\Rightarrow$ $f(1) = 0$
> 
> $$f(1) = 1 + a + 9 + b = 0$$
> 
> $$1 - 6 + 9 + b = 0$$
> 
> $$b = -4$$
> 
> **답**: $ab = (-6) \times (-4) = 24$

### 예제 210

함수 $f(x) = x^3 - 3x$에 대한 <보기>의 설명 중 옳은 것을 모두 골라라.

**(가)** $f(x)$는 극댓값과 극솟값을 가진다.
**(나)** $x \geq 2$이면 $f(x) \geq 2$이다.
**(다)** $|x| \leq 2$이면 $|f(x)| \leq 2$이다.

> [!summary]- 풀이
> **(가) 극값 확인**:
> 
> $$f'(x) = 3x^2 - 3 = 3(x^2 - 1) = 3(x - 1)(x + 1) = 0$$
> 
> $$x = -1 \quad \text{or} \quad x = 1$$
> 
> $$f''(x) = 6x$$
> 
> $$f''(-1) = -6 < 0 \quad \Rightarrow \quad x = -1 \text{에서 극대}$$
> 
> $$f''(1) = 6 > 0 \quad \Rightarrow \quad x = 1 \text{에서 극소}$$
> 
> 극값 계산:
> - 극댓값: $f(-1) = -1 + 3 = 2$
> - 극솟값: $f(1) = 1 - 3 = -2$
> 
> $\therefore$ **(가)는 참**
> 
> ---
> 
> **(나) $x \geq 2$일 때 $f(x) \geq 2$인지 확인**:
> 
> $x = 1$에서 극소이고, $x > 1$에서 $f'(x) > 0$이므로 증가함수
> 
> $$f(2) = 8 - 6 = 2$$
> 
> $x \geq 2$이면 $f(x) \geq f(2) = 2$
> 
> $\therefore$ **(나)는 참**
> 
> ---
> 
> **(다) $|x| \leq 2$일 때 $|f(x)| \leq 2$인지 확인**:
> 
> 구간 $[-2, 2]$에서 함수값 범위:
> - $f(-2) = -8 + 6 = -2$
> - $f(-1) = 2$ (극댓값)
> - $f(1) = -2$ (극솟값)
> - $f(2) = 2$
> 
> 최댓값: $2$, 최솟값: $-2$
> 
> 따라서 $-2 \leq f(x) \leq 2$ $\Rightarrow$ $|f(x)| \leq 2$
> 
> $\therefore$ **(다)는 참**
> 
> ---
> 
> **답**: (가), (나), (다) 모두 옳다

### 예제 211

함수 $f(x) = x^4 + ax^3 + bx^2 + cx + 6$이 다음 조건을 만족시킬 때 $f(3)$의 값을 구하여라.

**(가)** 모든 실수 $x$에 대하여 $f(-x) = f(x)$
**(나)** $f(x)$의 극솟값이 $-10$을 갖는다.

> [!summary]- 풀이
> **조건 (가) 분석**: $f(-x) = f(x)$ $\Rightarrow$ 우함수
> 
> 우함수는 홀수 차수 항의 계수가 $0$
> 
> $$a = 0, \quad c = 0$$
> 
> $$f(x) = x^4 + bx^2 + 6$$
> 
> **조건 (나) 분석**: 극값 찾기
> 
> $$f'(x) = 4x^3 + 2bx = 2x(2x^2 + b) = 0$$
> 
> $$x = 0 \quad \text{or} \quad 2x^2 + b = 0$$
> 
> **경우 1**: $b > 0$일 때
> 
> $2x^2 + b = 0$이 실근을 갖지 않으므로 극점은 $x = 0$ 하나
> 
> $$f(0) = 6 \neq -10$$
> 
> 모순 $\therefore$ $b > 0$은 불가능
> 
> **경우 2**: $b < 0$일 때
> 
> $$2x^2 + b = 0 \quad \Rightarrow \quad x = \pm\sqrt{-\frac{b}{2}}$$
> 
> 극점: $x = 0, \pm\sqrt{-\frac{b}{2}}$
> 
> $f''(x) = 12x^2 + 2b$에서
> - $f''(0) = 2b < 0$ $\Rightarrow$ $x = 0$에서 극대
> - $f''\left(\pm\sqrt{-\frac{b}{2}}\right) = 12 \cdot \left(-\frac{b}{2}\right) + 2b = -4b > 0$ $\Rightarrow$ 극소
> 
> 극솟값 계산:
> 
> $$f\left(\pm\sqrt{-\frac{b}{2}}\right) = \left(-\frac{b}{2}\right)^2 + b \cdot \left(-\frac{b}{2}\right) + 6$$
> 
> $$= \frac{b^2}{4} - \frac{b^2}{2} + 6 = -\frac{b^2}{4} + 6$$
> 
> 극솟값이 $-10$:
> 
> $$-\frac{b^2}{4} + 6 = -10$$
> 
> $$-\frac{b^2}{4} = -16$$
> 
> $$b^2 = 64$$
> 
> $$b = \pm 8$$
> 
> $b < 0$이므로 **$b = -8$**
> 
> **답 계산**:
> 
> $$f(x) = x^4 - 8x^2 + 6$$
> 
> $$f(3) = 81 - 72 + 6 = 15$$

### 예제 212

최고차 항의 계수가 양인 사차함수 $f(x)$에 대하여 $f'(x) = 0$는 서로 다른 세 실근 $\alpha, \beta, \gamma$ ($\alpha < \beta < \gamma$)를 갖고 $f(\alpha)f(\beta)f(\gamma) < 0$일 때 다음 <보기> 중 옳은 것을 모두 골라라.

**(가)** 함수 $f(x)$는 $x = \beta$에서 극댓값을 갖는다.
**(나)** 방정식 $f(x) = 0$는 서로 다른 두 실근을 갖는다.
**(다)** $f(\alpha) > 0$이면 방정식 $f(x) = 0$는 $\beta$보다 작은 실근을 가진다.

> [!summary]- 풀이
> **상황 분석**:
> 
> 사차함수의 최고차항 계수가 양이고 $f'(x) = 0$이 서로 다른 세 실근을 가짐
> 
> $\Rightarrow$ 극값이 3개: 극소 2개, 극대 1개
> 
> $\alpha < \beta < \gamma$이므로:
> - $x = \alpha$: 극소
> - $x = \beta$: 극대
> - $x = \gamma$: 극소
> 
> **조건**: $f(\alpha)f(\beta)f(\gamma) < 0$
> 
> 세 극값의 곱이 음수 $\Rightarrow$ 홀수 개가 음수
> 
> 가능한 경우:
> 1. $f(\alpha) < 0, f(\beta) > 0, f(\gamma) > 0$ (극소 1개가 음, 극대와 극소 1개가 양)
> 2. $f(\alpha) > 0, f(\beta) < 0, f(\gamma) > 0$ (극대가 음)
> 3. $f(\alpha) > 0, f(\beta) > 0, f(\gamma) < 0$ (극소 1개가 음)
> 
> <!-- 그래프: 사차함수의 극값 배치 -->
> ```
>    경우 1:              경우 2:              경우 3:
>       ●                                       ●
>      / \                    ─●─              / \
>     /   \                  /   \            /   \
>    ●─────              ●       ●        ────     ●─
> ```
> 
> ---
> 
> **(가) $x = \beta$에서 극댓값?**
> 
> 위 3가지 경우 모두 $\beta$는 극대점
> 
> $\therefore$ **(가)는 참**
> 
> ---
> 
> **(나) $f(x) = 0$는 서로 다른 두 실근?**
> 
> 3가지 경우 모두 $x$축과 2개의 점에서 접함 (중근 2개)
> 
> 서로 다른 두 실근을 가짐
> 
> $\therefore$ **(나)는 참**
> 
> ---
> 
> **(다) $f(\alpha) > 0$이면 $f(x) = 0$이 $\beta$보다 작은 실근?**
> 
> $f(\alpha) > 0$인 경우: 경우 2, 경우 3
> 
> - **경우 2**: $f(\beta) < 0$ $\Rightarrow$ 두 실근이 모두 $\beta$보다 오른쪽
> - **경우 3**: $f(\gamma) < 0$ $\Rightarrow$ 두 실근 중 하나는 $\beta$보다 왼쪽, 하나는 오른쪽
> 
> 항상 $\beta$보다 작은 실근을 갖는 것은 아님
> 
> $\therefore$ **(다)는 거짓**
> 
> ---
> 
> **답**: (가), (나)

### 예제 213

$a > 1$일 때 함수 $f(x) = 2x^3 - 3(a+1)x^2 + 6ax - 4a + 2$에 대하여 방정식 $f(x) = 0$의 한 실근을 $b$라 하자. 다음은 두 수 $a, b$의 크기를 비교하는 과정이다. ??을 채우시오.

$f'(a) = $ ?? 이고 $a > 1$이므로 $f(x)$는 $x = 1$에서 ??을 가진다.
그런데 $f(1) < 0$이고 $f(b) = 0$이므로 $a$ ?? $b$이다.

> [!summary]- 풀이
> **(1) $f'(x)$ 계산**:
> 
> $$f'(x) = 6x^2 - 6(a+1)x + 6a$$
> 
> $$= 6[x^2 - (a+1)x + a]$$
> 
> $$= 6(x - 1)(x - a)$$
> 
> **답**: $f'(x) = 6(x-1)(x-a)$
> 
> ---
> 
> **(2) $x = 1$에서의 극값 판정**:
> 
> $f'(x) = 0$의 두 실근: $x = 1, x = a$ (단, $a > 1$)
> 
> 삼차항의 계수가 양수이므로:
> - $x = 1$: 극대
> - $x = a$: 극소
> 
> **답**: 극대
> 
> ---
> 
> **(3) $a$와 $b$의 대소 비교**:
> 
> $$f(1) = 2 - 3(a+1) + 6a - 4a + 2 = 2 - 3a - 3 + 2a + 2 = 1 - a < 0$$
> 
> ($a > 1$이므로 $f(1) < 0$)
> 
> $x = 1$에서 극댓값이 음수 $\Rightarrow$ $x = a$에서 극솟값도 음수
> 
> 따라서 $x < a$ 구간에서 함수값은 모두 음수
> 
> $f(b) = 0$이려면 $b > a$
> 
> **답**: $a < b$

### 예제 214

함수 $f(x) = x^3 - 6x^2 + 9x + 1$의 극댓값과 극솟값의 차를 구하시오.

> [!summary]- 풀이
> **극점 찾기**:
> 
> $$f'(x) = 3x^2 - 12x + 9 = 3(x^2 - 4x + 3) = 3(x - 1)(x - 3) = 0$$
> 
> $$x = 1 \quad \text{or} \quad x = 3$$
> 
> **극값 판정**:
> 
> 삼차항의 계수가 양수이므로:
> - $x = 1$: 극대
> - $x = 3$: 극소
> 
> **극값 계산**:
> 
> $$f(1) = 1 - 6 + 9 + 1 = 5$$
> 
> $$f(3) = 27 - 54 + 27 + 1 = 1$$
> 
> **극댓값과 극솟값의 차**:
> 
> $$|f(1) - f(3)| = |5 - 1| = 4$$

### 예제 215

함수 $f(x) = 2x^3 - 3x^2 + 6ax$ $\left(a < \frac{1}{4}\right)$가 충분히 작은 양수 $h$에 대하여

**(가)** $f(\alpha - h) < f(\alpha), \quad f(\alpha) > f(\alpha + h)$
**(나)** $f(\beta - h) > f(\beta), \quad f(\beta) < f(\beta + h)$

를 만족시키는 $\alpha, \beta$에 대하여 $|f(\alpha) - f(\beta)| = 8$이라고 한다. 이때 $a + \alpha + \beta$의 값을 구하여라.

**힌트**: 삼차함수 $y = f(x)$에 대하여 $f'(x) = 0$의 서로 다른 두 실근을 $\alpha, \beta$라 하면

$$|f(\alpha) - f(\beta)| = \frac{|p|}{6}|\alpha - \beta|^3$$

(단, $f'(x) = px^2 + qx + r$이고, 이에 대한 증명은 적분 과정에서 배운다)

> [!summary]- 풀이
> **조건 분석**:
> 
> (가): $x = \alpha$ 좌우에서 $f(\alpha - h) < f(\alpha) > f(\alpha + h)$ $\Rightarrow$ 극대
> 
> (나): $x = \beta$ 좌우에서 $f(\beta - h) > f(\beta) < f(\beta + h)$ $\Rightarrow$ 극소
> 
> **도함수 계산**:
> 
> $$f'(x) = 6x^2 - 6x + 6a = 0$$
> 
> 두 근: $\alpha, \beta$
> 
> **근과 계수의 관계**:
> 
> $$\alpha + \beta = 1, \quad \alpha\beta = a$$
> 
> **힌트 공식 적용**:
> 
> $$|f(\alpha) - f(\beta)| = \frac{|p|}{6}|\alpha - \beta|^3 = \frac{6}{6}|\alpha - \beta|^3 = |\alpha - \beta|^3$$
> 
> 주어진 조건: $|f(\alpha) - f(\beta)| = 8$
> 
> $$|\alpha - \beta|^3 = 8$$
> 
> $$|\alpha - \beta| = 2$$
> 
> **$a$ 구하기**:
> 
> $$|\alpha - \beta|^2 = (\alpha + \beta)^2 - 4\alpha\beta = 4$$
> 
> $$1^2 - 4a = 4$$
> 
> $$-4a = 3$$
> 
> $$a = -\frac{3}{4}$$
> 
> **답 계산**:
> 
> $$a + \alpha + \beta = -\frac{3}{4} + 1 = \frac{1}{4}$$

## 2. 극대-극소 차 공식 (참고)

삼차함수 $f(x)$에서 $f'(x) = px^2 + qx + r = 0$의 두 실근을 $\alpha, \beta$라 하면:

$$|f(\alpha) - f(\beta)| = \frac{|p|}{6}|\alpha - \beta|^3$$

**증명은 적분 단원에서 다룬다.**

이 공식은 삼차함수의 극댓값과 극솟값의 차를 계산할 때 유용하다.

## 연습문제

각 예제를 통해 학습한 내용을 복습하고, 다음을 숙지하시오:
1. 극값 판정: 증감표 또는 2계 도함수 이용
2. 극값 조건을 만족하는 미지수 결정: $f'(a) = 0$과 극값 조건 활용
3. 우함수 조건: $f(-x) = f(x)$ $\Rightarrow$ 홀수 차수 항의 계수 = 0
4. 사차함수의 극값 분석: 극값의 부호와 그래프 개형의 관계
5. 극대-극소 차 공식: $|f(\alpha) - f(\beta)| = \frac{|p|}{6}|\alpha - \beta|^3$
