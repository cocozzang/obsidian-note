# 다항함수의 함수방정식

## 1. 정리 (21): 다항함수의 함수방정식

함수 $f(x)$가 다음 세 가지 형태의 함수방정식 중 하나를 만족할 때, $f'(x)$를 구하는 두 가지 방법을 학습한다.

**방법 1**: 미분계수의 정의 이용
**방법 2**: 함수방정식을 $y$에 대해 편미분

> [!note] 편미분이란?
> 편미분을 간단히 설명하자면 여러 변수가 있는 함수에서 다른 변수들은 고정시키고 한 변수만 변화시켰을 때의 변화율을 구하는 것이다.

---

### 유형 ① $f(x+y) = f(x) + f(y) + C$

**함수방정식**: $f(x+y) = f(x) + f(y) + C

**Step 1**: $f(0)$ 구하기

$x = 0, y = 0$을 대입하면:

$$f(0) = f(0) + f(0) + C$$

$$\therefore f(0) = C$$

**Step 2-1**: 미분계수 정의를 이용한 방법

$$f'(x) = \lim_{y \to 0} \frac{f(x+y) - f(x)}{y}$$

$$= \lim_{y \to 0} \frac{f(x) + f(y) + C - f(x)}{y}$$

$$= \lim_{y \to 0} \frac{f(y) + C}{y}$$

$$= \lim_{y \to 0} \frac{f(y) - f(0)}{y} = f'(0)$$

**결론**: $f'(x) = f'(0)$ ($f'(x)=0$)

**Step 2-2**: 편미분을 이용한 방법

함수방정식 $f(x+y) = f(x) + f(y) + C$를 $y$에 대해 편미분하면:

$$f'(x+y) = f'(y)$$

$y = 0$을 대입하면:

$$f'(x) = f'(0)$$

---

강의에서는 f(x)가 1번 조건식을 만족할떄 1차함수라고 이야기 하는데
f'(0)=0인 경우에는 f(x)가 1번조건식을 만족하더라도 1차함수가 아니라
상수함수가 되는거같은데 잘 모르겠다.

f(x)가 아래 두조건을 만족하는 함수일떄
 조건1: $f'(0)=0$ , 조건2 : $f(x+y)=f(x)+f(y)-5$ 
 
 조건2의 식을 편미분하고 y=0을 대입하면 f'(x)=f'(0)=0
 이미 f'(x)가 0이므로 일차함수라고 이야기할 수없고,
 $f(x)=5$ 일떄 모든실수 범위 x,y에 대해 조건2의 조건식을 만족하는 걸로 보인다.

그러므로 f(x)가 조건식 $f(x+y)=f(x)+f(y)+c$를 만족하더라도
1차함수가 아니라 경우에 따라 상수함수가 될수있다 라고 생각됨.

---

### 유형 ② $f(x+y) = f(x) + f(y) + axy + C$

**함수방정식**: $f(x+y) = f(x) + f(y) + axy + C$

**Step 1**: $f(0) = C$

**Step 2-1**: 미분계수 정의를 이용한 방법

$$f'(x) = \lim_{y \to 0} \frac{f(x+y) - f(x)}{y}$$

$$= \lim_{y \to 0} \frac{f(x) + f(y) + axy + C - f(x)}{y}$$

$$= \lim_{y \to 0} \frac{f(y) + C}{y} + \frac{axy}{y}$$

$$= f'(0) + ax$$

**결론**: $f'(x) = f'(0) + ax$

**Step 2-2**: 편미분을 이용한 방법

함수방정식을 $y$에 대해 편미분하면:

$$f'(x+y) = f'(y) + ax$$

$y = 0$을 대입하면:

$$f'(x) = f'(0) + ax$$

---

### 유형 ③ $f(x+y) = f(x) + f(y) + axy(x+y+b) + C$

**함수방정식**: $f(x+y) = f(x) + f(y) + axy(x+y+b) + C$

**Step 1**: $f(0) = C$

**Step 2-1**: 미분계수 정의를 이용한 방법

$$f'(x) = \lim_{y \to 0} \frac{f(x+y) - f(x)}{y}$$

$$= \lim_{y \to 0} \frac{f(x) + f(y) + axy(x+y+b) + C - f(x)}{y}$$

$$= \lim_{y \to 0} \frac{f(y) + ax^2y + axy^2 + abxy + C}{y}$$

$$= \lim_{y \to 0} \frac{f(y) + C}{y} + \frac{ax^2y + axy^2 + abxy}{y}$$

$$= f'(0) + ax^2 + abx$$

**결론**: $f'(x) = f'(0) + ax^2 + abx$

**Step 2-2**: 편미분을 이용한 방법

함수방정식을 $y$에 대해 편미분하면:

$$f(x+y) = f(x) + f(y) + axy(x+y+b) + C$$

$$f'(x+y) = f'(y) + ax^2 + 2axy + abx$$

$y = 0$을 대입하면:

$$f'(x) = f'(0) + ax^2 + abx$$

## 2. 예제

### 예제 109

모든 실수 $x$에 대하여 정의된 함수 $f(x)$가 임의의 실수 $x, y$에 대하여 조건식 $f(x+y) = f(x) + f(y) + xy$를 만족하고 $f'(0) = 5$일 때 $f'(1)$을 구하여라.

> [!summary]- 풀이
> 주어진 함수방정식: $f(x+y) = f(x) + f(y) + xy$
> 
> 이는 유형 ②에 해당한다. ($a = 1, C = 0$)
> 
> **방법 1**: 조건식을 $y$로 편미분
> 
> $$f'(x+y) = f'(y) + x$$
> 
> $y = 0$을 대입하면:
> 
> $$f'(x) = f'(0) + x = x + 5$$
> 
> 따라서:
> 
> $$f'(1) = 1 + 5 = 6$$

### 예제 110

다항함수 $f(x)$는 모든 실수 $x, y$에 대하여 조건식 $f(x+y) = f(x) + f(y) + 2xy - 1$을 만족시킨다. 주어진 식 $\lim_{x \to 1} \frac{f(x) - f'(x)}{x^2 - 1} = 14$일 때 $f'(0)$의 값을 구하라.

> [!summary]- 풀이
> **Step 1**: 주어진 극한식 분석
> 
> 극한의 정의를 통해 $f(1) = f'(1)$임을 알 수 있다.
> 
> **Step 2**: 조건식 편미분
> 
> $f(x+y) = f(x) + f(y) + 2xy - 1$을 $y$로 편미분:
> 
> $$f'(x+y) = f'(y) + 2x$$
> 
> $y = 0$을 대입:
> 
> $$f'(x) = f'(0) + 2x$$
> 
> **Step 3**: 극한식 정리
> 
> $$\lim_{x \to 1} \frac{f(x) - f'(x)}{x^2 - 1} = 14$$
> 
> $$\lim_{x \to 1} \frac{f(x) - f(1)}{(x-1)(x+1)} - \frac{f'(x) - f(1)}{(x-1)(x+1)} = 14$$
> 
> $$\lim_{x \to 1} \frac{1}{x+1} \left( \frac{f(x) - f(1)}{x - 1} - \frac{f'(x) - f'(1)}{x - 1} \right) = 14$$
> 
> $$\lim_{x \to 1} \frac{1}{2} \left( f'(1) - f''(1) \right) = 14$$
> 
> $$f'(1) - f''(1) = 28$$
> 
> **Step 4**: $f''(x)$ 구하기
> 
> $f'(x) = f'(0) + 2x$를 미분하면:
> 
> $$f''(x) = 2$$
> 
> **Step 5**: 계산
> 
> $$f'(1) - 2 = 28$$
> 
> $$f'(1) = 30$$
> 
> 그런데 $f'(1) = f'(0) + 2 \cdot 1 = f'(0) + 2$이므로:
> 
> $$f'(0) + 2 = 30$$
> 
> $$\therefore f'(0) = 28$$

### 예제 111

실수에서 정의된 미분 가능한 함수 $f(x)$가 다음 두 조건을 만족한다.

① 임의의 실수 $x, y$에 대하여 $f(x-y) = f(x) - f(y) + xy(x-y)$
② $f'(0) = 8$

함수 $f(x)$가 $x = a$에서 극댓값을 갖고 $x = b$에서 극솟값을 가질 때 $a^2 + b^2$의 값을 구하시오.

> [!summary]- 풀이
> **Step 1**: 극값 조건
> 
> $f(x)$가 $x = a$에서 극댓값, $x = b$에서 극솟값을 가진다는 것은:
> 
> $$f'(a) = 0, \quad f'(b) = 0$$
> 
> **Step 2**: 조건식을 $y$에 대하여 미분
> 
> $f(x-y) = f(x) - f(y) + xy(x-y)$를 $y$로 편미분:
> 
> $$-f'(x-y) = -f'(y) + x^2 - 2xy$$
> 
> $y = 0$을 대입:
> 
> $$-f'(x) = -f'(0) + x^2$$
> 
> $$f'(x) = f'(0) - x^2 = 8 - x^2$$
> 
> **Step 3**: 극값 조건 적용
> 
> $f'(a) = 0$:
> 
> $$8 - a^2 = 0 \quad \Rightarrow \quad a^2 = 8$$
> 
> $f'(b) = 0$:
> 
> $$8 - b^2 = 0 \quad \Rightarrow \quad b^2 = 8$$
> 
> **Step 4**: 답 구하기
> 
> $$a^2 + b^2 = 8 + 8 = 16$$

### 예제 112

함수 $y = f(x)$가 모든 실수 $x, y$에 대하여 $f(x+y) = f(x) + f(y) + 5xy$를 만족하고 $x = 0$에서의 접선의 기울기가 $1$일 때 $f(x)$의 최솟값을 $m$이라 한다. $-100m$의 값을 구하여라.

> [!note] 간단한 적분 방법
> $f'(x)$를 적분하면 $f(x)$를 구할 수 있다. 이때 부정적분상수는 함수방정식에 $x = 0, y = 0$을 대입하여 결정한다.
> 
> 예: $f(x+y) = f(x) + f(y) + 2xy$, $f'(0) = 2$일 때
> - $f'(x) = f'(0) + 2x = 2x + 2$를 적분
> - $f(x) = x^2 + 2x + C$
> - $x = 0, y = 0$ 대입: $f(0) = f(0) + f(0)$ → $f(0) = 0$
> - $\therefore f(x) = x^2 + 2x$

> [!summary]- 풀이
> **Step 1**: 조건 해석
> 
> $x = 0$에서의 접선의 기울기가 $1$이라는 것은:
> 
> $$f'(0) = 1$$
> 
> **Step 2**: $f'(x)$ 구하기
> 
> $f(x+y) = f(x) + f(y) + 5xy$를 $y$에 대해 편미분:
> 
> $$f'(x+y) = f'(y) + 5x$$
> 
> $y = 0$을 대입:
> 
> $$f'(x) = f'(0) + 5x = 5x + 1$$
> 
> **Step 3**: $f(x)$의 성질 분석
> 
> $f'(x) = 5x + 1$은 일차함수이므로 $f(x)$는 이차함수다.
> 
> $f'(x) = 0$일 때: $x = -\frac{1}{5}$
> 
> $f'(0) = 1 > 0$이므로 $f(x)$는 아래로 볼록한 이차함수다.
> 
> $$\therefore m = f\left(-\frac{1}{5}\right)$$
> 
> **Step 4**: $f(x)$ 구하기
> 
> $f'(x) = 5x + 1$을 적분:
> 
> $$f(x) = \frac{5}{2}x^2 + x + C$$
> 
> $x = 0, y = 0$을 함수방정식에 대입:
> 
> $$f(0) = f(0) + f(0) + 0$$
> 
> $$f(0) = 0$$
> 
> 따라서 부정적분상수 $C = 0$:
> 
> $$f(x) = \frac{5}{2}x^2 + x$$
> 
> **Step 5**: 최솟값 계산
> 
> $$m = f\left(-\frac{1}{5}\right) = \frac{5}{2} \cdot \frac{1}{25} - \frac{1}{5}$$
> 
> $$= \frac{1}{10} - \frac{2}{10} = -\frac{1}{10}$$
> 
> $$-100m = -100 \cdot \left(-\frac{1}{10}\right) = 10$$

### 예제 113

항상 양의 값을 갖는 미분 가능한 함수 $f(x)$가 모든 실수 $x, y$에 대하여 $f(x+y) = 2f(x)f(y)$를 만족시키고 $f'(0) = 1$일 때 $\frac{f'(10)}{f(10)}$을 구하여라.

> [!summary]- 풀이
> **Step 1**: 함수방정식 편미분
> 
> $f(x+y) = 2f(x)f(y)$를 $y$에 대해 편미분:
> 
> $$f'(x+y) = 2f(x)f'(y)$$
> 
> **Step 2**: $y = 0$ 대입
> 
> $$f'(x) = 2f(x)f'(0) = 2f(x) \cdot 1 = 2f(x)$$
> 
> **Step 3**: 계산
> 
> $$\frac{f'(10)}{f(10)} = \frac{2f(10)}{f(10)} = 2$$

### 예제 114

미분 가능한 두 함수 $f(x), g(x)$가 모든 실수 $x, y$에 대하여 $f(x+y) = f(x) + g(2y)$를 만족시키고 $g'(0) = 4$일 때 $f'(10)$을 구하여라.

> [!summary]- 풀이
> **Step 1**: 함수방정식 편미분
> 
> $f(x+y) = f(x) + g(2y)$를 $y$에 대해 편미분:
> 
> $$f'(x+y) = 2g'(2y)$$
> 
> **Step 2**: $y = 0$ 대입
> 
> $$f'(x) = 2g'(0) = 2 \cdot 4 = 8$$
> 
> **Step 3**: 결론
> 
> $f(x)$는 모든 구간에서 기울기가 $8$인 일차함수이므로:
> 
> $$f'(10) = 8$$
