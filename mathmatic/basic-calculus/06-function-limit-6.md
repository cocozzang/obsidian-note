# 함수의 극한(6) - 불연속점의 개수

## Thm (9): 불연속점의 개수

## $\lim_{n \to \infty} x^n$의 극한

$x$의 값에 따라 $\lim_{n \to \infty} x^n$의 수렴/발산이 결정된다:

### 경우 분류

| 조건          | 극한                             | 비고  |
| ----------- | ------------------------------ | --- |
| $\|x\| < 1$ | $\lim_{n \to \infty} x^n = 0$  | 수렴  |
| $\|x\| > 1$ | $\lim_{n \to \infty} x^n$ (발산) | 발산  |
| $x = 1$     | $\lim_{n \to \infty} x^n = 1$  | 수렴  |
| $x = -1$    | $\lim_{n \to \infty} x^n$ (진동) | 진동  |

### 상세 설명

**① $|x| < 1$인 경우:**

- $-1 < x < 1$일 때, $x^n$은 $n \to \infty$일 때 0으로 수렴
- 예: $x = 0.5$이면 $(0.5)^n \to 0$

**② $|x| > 1$인 경우:**

- $x > 1$ 또는 $x < -1$일 때, $|x^n| \to \infty$로 발산
- 예: $x = 2$이면 $2^n \to \infty$
- 예: $x = -2$이면 $(-2)^n$은 부호가 바뀌며 절댓값이 무한대로 발산

**③ $x = 1$인 경우:**

- $1^n = 1$ (모든 $n$에 대해)
- $\lim_{n \to \infty} 1^n = 1$

**④ $x = -1$인 경우:**

- $(-1)^n$은 $n$이 짝수면 1, 홀수면 -1
- 진동하므로 극한이 존재하지 않음

## 예제 33

함수 $f(x) = \lim_{n \to \infty} \frac{x^2 - x^{n+2}}{1 + x^n}$의 불연속이 되는 $x$의 값의 개수는 몇 개인가?

(단, $n$은 양의 정수)

> [!summary]- 풀이
> $x$의 값에 따라 $\lim_{n \to \infty} x^n$의 값이 달라지므로, 경우를 나누어 $f(x)$를 구한다.
>
> **경우 1: $|x| < 1$**
>
> $\lim_{n \to \infty} x^n = 0$이므로:
> $$f(x) = \lim_{n \to \infty} \frac{x^2 - x^{n+2}}{1 + x^n} = \frac{x^2 - 0}{1 + 0} = x^2$$
>
> **경우 2: $x = 1$**
>
> $\lim_{n \to \infty} x^n = 1$이므로:
> $$f(1) = \lim_{n \to \infty} \frac{1 - 1}{1 + 1} = \frac{0}{2} = 0$$
>
> **경우 3: $x = -1$**
>
> $x^n$이 진동하므로 극한이 존재하지 않음. 따라서 $f(-1)$은 정의되지 않음.
>
> **경우 4: $|x| > 1$**
>
> $|x^n| \to \infty$이므로, 분자와 분모를 $x^n$으로 나누면:
> $$f(x) = \lim_{n \to \infty} \frac{\frac{x^2}{x^n} - x^2}{\frac{1}{x^n} + 1}$$
>
> $\frac{x^2}{x^n} = \frac{1}{x^{n-2}} \to 0$ (as $n \to \infty$)
>
> $$f(x) = \frac{0 - x^2}{0 + 1} = -x^2$$
>
> **함수 정리:**
>
> $$
> f(x) = \begin{cases}
> x^2 & (|x| < 1) \\
> 0 & (x = 1) \\
> \text{정의 안됨} & (x = -1) \\
> -x^2 & (|x| > 1)
> \end{cases}
> $$
>
> **불연속점 판정:**
>
> **$x = 1$에서:**
>
> - 좌극한: $\lim_{x \to 1^-} f(x) = 1^2 = 1$
> - 함수값: $f(1) = 0$
> - 좌극한 ≠ 함수값이므로 **불연속**
>
> **$x = -1$에서:**
>
> - $f(-1)$이 정의되지 않으므로 **불연속**
>
> 따라서 불연속점은 $x = 1, -1$로 **2개**
>
> **답:** $\boxed{2}$

## 예제 34

함수 $f(x) = \lim_{n \to \infty} \frac{(2x)^{2n-1} + 1}{(2x)^{2n} + 1}$의 불연속이 되는 $x$의 값을 구하여라.

> [!summary]- 풀이
> $(2x)^{2n-1} = (2x)^{2n} \cdot \frac{1}{2x} = (2x)^{2n} / (2x)$
>
> $t = (2x)^2 = 4x^2$로 놓으면, $(2x)^{2n} = t^n$
>
> $$f(x) = \lim_{n \to \infty} \frac{t^n / (2x) + 1}{t^n + 1}$$
>
> $t = 4x^2$의 값에 따라 경우를 나눈다.
>
> **경우 1: $|t| < 1$ 즉, $4x^2 < 1$ 즉, $|x| < \frac{1}{2}$**
>
> $\lim_{n \to \infty} t^n = 0$이므로:
> $$f(x) = \frac{0 + 1}{0 + 1} = 1$$
>
> **경우 2: $t = 1$ 즉, $4x^2 = 1$ 즉, $x = \pm\frac{1}{2}$**
>
> $\lim_{n \to \infty} t^n = 1$이므로:
> $$f\left(\pm\frac{1}{2}\right) = \lim_{n \to \infty} \frac{1/(2 \cdot \pm\frac{1}{2}) + 1}{1 + 1}$$
>
> $x = \frac{1}{2}$: $f\left(\frac{1}{2}\right) = \frac{1/1 + 1}{2} = \frac{2}{2} = 1$
>
> $x = -\frac{1}{2}$: $f\left(-\frac{1}{2}\right) = \frac{1/(-1) + 1}{2} = \frac{0}{2} = 0$
>
> **경우 3: $|t| > 1$ 즉, $|x| > \frac{1}{2}$**
>
> 분자와 분모를 $t^n$으로 나누면:
> $$f(x) = \lim_{n \to \infty} \frac{1/(2x) + 1/t^n}{1 + 1/t^n} = \frac{1/(2x) + 0}{1 + 0} = \frac{1}{2x}$$
>
> **함수 정리:**
>
> $$
> f(x) = \begin{cases}
> 1 & (|x| < \frac{1}{2}) \\
> 1 & (x = \frac{1}{2}) \\
> 0 & (x = -\frac{1}{2}) \\
> \frac{1}{2x} & (|x| > \frac{1}{2})
> \end{cases}
> $$
>
> **불연속점 판정:**
>
> **$x = \frac{1}{2}$에서:**
>
> - 좌극한: $\lim_{x \to (1/2)^-} f(x) = 1$
> - 우극한: $\lim_{x \to (1/2)^+} f(x) = \frac{1}{2 \cdot 1/2} = 1$
> - 함수값: $f(1/2) = 1$
> - 연속
>
> **$x = -\frac{1}{2}$에서:**
>
> - 좌극한: $\lim_{x \to (-1/2)^-} f(x) = \frac{1}{2 \cdot (-1/2)} = -1$
> - 우극한: $\lim_{x \to (-1/2)^+} f(x) = 1$
> - 함수값: $f(-1/2) = 0$
> - 좌극한 ≠ 우극한이므로 **불연속**
>
> **$x = 0$에서:**
>
> - $f(0) = 1$ (왼쪽 식)
> - 우극한에서 $\frac{1}{2x}$ 형태는 $x = 0$에서 정의되지 않지만, $|x| < 1/2$ 범위에서는 $f(x) = 1$
> - 실제로 $x = 0$은 $|x| < 1/2$ 범위에 속하므로 $f(0) = 1$
> - 연속
>
> 따라서 불연속점은 $x = -\frac{1}{2}$
>
> **답:** $\boxed{x = -\frac{1}{2}}$

## 예제 35

함수 $f(x) = \lim_{n \to \infty} \frac{x^{2n+1} + ax + b}{x^{2n} + 1}$가 모든 실수 $x$에 대하여 연속이 되도록 상수 $a$, $b$의 값을 정할 때 $3a + b$의 값을 구하여라.

> [!summary]- 풀이
> $t = x^2$로 놓으면 $x^{2n} = t^n$이고, $x^{2n+1} = x \cdot t^n$
>
> $$f(x) = \lim_{n \to \infty} \frac{x \cdot t^n + ax + b}{t^n + 1}$$
>
> $t = x^2$의 값에 따라 경우를 나눈다.
>
> **경우 1: $|t| < 1$ 즉, $x^2 < 1$ 즉, $|x| < 1$**
>
> $\lim_{n \to \infty} t^n = 0$이므로:
> $$f(x) = \frac{0 + ax + b}{0 + 1} = ax + b$$
>
> **경우 2: $t = 1$ 즉, $x^2 = 1$ 즉, $x = \pm 1$**
>
> $\lim_{n \to \infty} t^n = 1$이므로:
> $$f(\pm 1) = \frac{(\pm 1) \cdot 1 + a(\pm 1) + b}{1 + 1}$$
>
> $x = 1$: $f(1) = \frac{1 + a + b}{2}$
>
> $x = -1$: $f(-1) = \frac{-1 - a + b}{2}$
>
> **경우 3: $|t| > 1$ 즉, $|x| > 1$**
>
> 분자와 분모를 $t^n$으로 나누면:
> $$f(x) = \lim_{n \to \infty} \frac{x + (ax+b)/t^n}{1 + 1/t^n} = \frac{x + 0}{1 + 0} = x$$
>
> **연속 조건:**
>
> **$x = 1$에서 연속:**
> $$\lim_{x \to 1^-} f(x) = a \cdot 1 + b = a + b$$
> $$\lim_{x \to 1^+} f(x) = 1$$
> $$f(1) = \frac{1 + a + b}{2}$$
>
> 연속이려면:
> $$a + b = \frac{1 + a + b}{2} = 1$$
>
> 첫 번째 조건: $a + b = 1$ ... ①
>
> 두 번째 조건: $\frac{1 + a + b}{2} = 1$
> $$1 + a + b = 2$$
> $$a + b = 1$$ (①과 일치)
>
> **$x = -1$에서 연속:**
> $$\lim_{x \to (-1)^-} f(x) = -1$$
> $$\lim_{x \to (-1)^+} f(x) = a(-1) + b = -a + b$$
> $$f(-1) = \frac{-1 - a + b}{2}$$
>
> 연속이려면:
> $$-1 = -a + b = \frac{-1 - a + b}{2}$$
>
> 첫 번째 조건: $-a + b = -1$ ... ②
>
> 두 번째 조건: $\frac{-1 - a + b}{2} = -1$
> $$-1 - a + b = -2$$
> $$-a + b = -1$$ (②와 일치)
>
> **연립방정식 풀이:**
>
> ① $a + b = 1$  
> ② $-a + b = -1$
>
> ①에서 ②를 빼면:
> $$(a + b) - (-a + b) = 1 - (-1)$$
> $$2a = 2$$
> $$a = 1$$
>
> ①에 대입:
> $$1 + b = 1$$
> $$b = 0$$
>
> 따라서:
> $$3a + b = 3(1) + 0 = 3$$
>
> **답:** $\boxed{3}$

## 예제 36

삼차함수 $y = f(x)$가 극댓값 $\frac{1}{2}$, 극솟값 $-2$를 가질 때 함수 $g(x)$를 다음과 같이 정의한다.

$$g(x) = \lim_{n \to \infty} \frac{1}{1 + \{f(x)\}^{2n}}$$

이 때 실수 전체의 집합에서 함수 $y = g(x)$는 $x = \alpha$에서 불연속이다. $\alpha$의 개수는 몇 개인가?

> [!summary]- 풀이
> $t = f(x)$로 놓으면:
> $$g(x) = \lim_{n \to \infty} \frac{1}{1 + t^{2n}}$$
>
> $t^{2n} = (t^2)^n$이므로, $s = t^2 = \{f(x)\}^2$로 놓으면:
> $$g(x) = \lim_{n \to \infty} \frac{1}{1 + s^n}$$
>
> **경우 1: $|s| < 1$ 즉, $|t| < 1$ 즉, $|f(x)| < 1$**
>
> $\lim_{n \to \infty} s^n = 0$이므로:
> $$g(x) = \frac{1}{1 + 0} = 1$$
>
> **경우 2: $s = 1$ 즉, $|t| = 1$ 즉, $|f(x)| = 1$**
>
> $\lim_{n \to \infty} s^n = 1$이므로:
> $$g(x) = \frac{1}{1 + 1} = \frac{1}{2}$$
>
> **경우 3: $|s| > 1$ 즉, $|t| > 1$ 즉, $|f(x)| > 1$**
>
> 분자와 분모를 $s^n$으로 나누면:
> $$g(x) = \lim_{n \to \infty} \frac{1/s^n}{1/s^n + 1} = \frac{0}{0 + 1} = 0$$
>
> **함수 정리:**
>
> $$
> g(x) = \begin{cases}
> 1 & (|f(x)| < 1) \\
> \frac{1}{2} & (|f(x)| = 1) \\
> 0 & (|f(x)| > 1)
> \end{cases}
> $$
>
> **불연속점 판정:**
>
> $g(x)$가 불연속이 되는 점은 $f(x)$가 $\pm 1$을 지나는 점들이다.
>
> **삼차함수의 개형:**
>
> - 극댓값: $\frac{1}{2}$
> - 극솟값: $-2$
>
> 삼차함수는 극댓값과 극솟값 사이에서 감소하고, 양 끝에서 증가/감소한다.
>
> **$f(x) = 1$인 점:**
>
> - 극댓값이 $\frac{1}{2} < 1$이므로, $f(x) = 1$은 극대 이전과 극대 이후에 각각 한 번씩 나타남
> - 총 2개의 점
>
> **$f(x) = -1$인 점:**
>
> - 극솟값이 $-2 < -1 < \frac{1}{2}$이므로, $f(x) = -1$은 극소 이전, 극소 이후에 각각 한 번씩 나타남
> - 총 2개의 점
>
> 따라서 불연속점은 총 **4개**
>
> **답:** $\boxed{4}$

## 참고사항

- **$x^n$의 극한:** $|x| < 1$이면 0, $x = 1$이면 1, $|x| > 1$이면 발산, $x = -1$이면 진동
- **극한 극한 함수:** $\lim_{n \to \infty}$가 포함된 함수는 $x$의 범위를 나누어 계산
- **불연속점 찾기:** 경계값($|x| = 1$ 등)에서 좌극한, 우극한, 함수값을 비교
- **그래프 활용:** 삼차함수의 극값을 이용하여 불연속점 개수 파악
