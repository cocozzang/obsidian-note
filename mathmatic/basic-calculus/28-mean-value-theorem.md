# 롤의 정리, 평균값 정리

## 1. 롤의 정리 (Rolle's Theorem)

### Thm (31): 롤의 정리

함수 $f(x)$가 폐구간 $[a, b]$에서 연속이고, 개구간 $(a, b)$에서 미분가능할 때 $f(a)=f(b)$이면 $f'(c)=0$ $(a<c<b)$를 만족시키는 $c$가 적어도 하나 존재한다.

### 증명

**i)** $[a, b]$에서 $f(x)$가 상수이면

모든 점에서 $f'(x)=0$이므로 $f'(c)=0$을 만족하는 점 $c(a<c<b)$가 적어도 하나 존재한다.

**ii)** $f(x)$가 상수함수가 아니면

$f(x_0)=f(a)$ $(∵ f(a)=f(b))$인 점 $x_0$ $(a<x_0<b)$가 존재한다.

또, [[07-function-limit-7#Thm (12) 최대, 최소의 정리|Thm(12)]] 최대최소정리에 의해 $f$는 $[a, b]$에서 최댓값, 최솟값을 가진다.

그런데 $f(a)=f(b)$이므로 어떤 점 $c$ $(a<c<b)$에서 최댓값 $f(c)$를 가지면 $f(c)$는 극댓값이므로 페르마 정리에 의해 $f'(c)=0$이다.

또, 어떤 점 $c$ $(a<c<b)$에서 최솟값 $f(c)$를 가지면 $f(c)$는 극솟값이므로 페르마 정리에 의해 $f'(c)=0$이다.

<!-- 그래프: 롤의 정리의 기하학적 의미 -->
```

    |                         |                           |
    |    f'(c)=0              |     f'(c)=0               |     f'(c₁)=0  f'(c₂)=0
    |       ●                 |        ●●●                |        ●         ●
    |      / \                |       /   \               |       / \       / \

    |....●.....●.......       |.....●.......●.....        |.....●.....●...●.....●....
    |    a  c  b              |     a   c   b             |     a    c₁   c₂    b
    +----------------         +------------------         +----------------------
```

따라서, $f'(c)=0$ $(a<c<b)$를 만족하는 점 $c$ $(a<c<b)$가 적어도 하나 존재한다.

### 예제 167

$f(x)=k(x-a)(x-b)$는 롤의 정리가 성립함을 보이고 롤의 정리를 만족하는 $c$ $(a<c<b)$를 구하여라.

> [!summary]- 풀이
> $f(x)=k(x-a)(x-b)$는 이차함수이므로 모든 실수에서 미분가능하다.
> 
> 특히, $f(a)=0$, $f(b)=0$이므로 $f(a)=f(b)$가 성립한다.
> 
> 따라서 롤의 정리가 성립한다.
> 
> 롤의 정리를 만족하는 $c$를 구하면:
> 
> $$f'(x)=k(x-b)+k(x-a)=k(2x-a-b)$$
> 
> $f'(c)=0$이므로
> 
> $$k(2c-a-b)=0$$
> 
> $$2c=a+b$$
> 
> $$\therefore c=\frac{a+b}{2}$$

## 2. 평균값 정리 (Mean Value Theorem)

### Thm (32): 평균값의 정리

함수 $f(x)$가 폐구간 $[a, b]$에서 연속이고 개구간 $(a, b)$에서 미분가능하면 $\frac{f(b)-f(a)}{b-a}=f'(c)$ $(a<c<b)$를 만족하는 $c$가 적어도 하나 존재한다.

### 증명

$$F(x)=f(x)-f(a)-\frac{f(b)-f(a)}{b-a}(x-a)$$

라 두면

$F(x)$는 $[a, b]$에서 연속이고 $(a, b)$에서 미분가능하며 $F(a)=F(b)$ $(=0)$이므로 [[#Thm (31) 롤의 정리|Thm(31)]] 롤의 정리에 의해 $F'(c)=0$인 $c$ $(a<c<b)$가 적어도 하나 존재한다.

그런데 

$$F'(x)=f'(x)-\frac{f(b)-f(a)}{b-a}$$

이므로 결국

$$F'(c)=f'(c)-\frac{f(b)-f(a)}{b-a}=0$$

$$\therefore \frac{f(b)-f(a)}{b-a}=f'(c)\ \ (a<c<b)$$

를 만족하는 $c$가 적어도 하나 존재한다.

<!-- 그래프: 평균값 정리의 기하학적 의미 -->
```
    y
    |
    |              f(c)=접선의 기울기
f(b)|...................●/
    |                 /|/
    |               /  /
    |             ●  /
f(a)|............/  /
    |          /  /
    |        /  /
    +---------●--------●-------- x
    O        a    c    b
```

**기하학적 의미**: 두 점 $(a, f(a))$와 $(b, f(b))$를 잇는 직선의 기울기 $\frac{f(b)-f(a)}{b-a}$와 평행한 접선이 $(a, b)$ 사이의 어떤 점 $c$에서 존재한다.

### 예제 168

폐구간 $[1, 2]$에서 함수 $f(x)=x^3$에 대하여 평균값 정리를 만족하는 $c$ $(1<c<2)$를 구하여라.

> [!summary]- 풀이
> 평균값 정리에 의해
> 
> $$\frac{f(2)-f(1)}{2-1}=f'(c)$$
> 
> $$f(2)=8,\ f(1)=1$$이므로
> 
> $$f'(c)=8-1=7$$
> 
> $f'(x)=3x^2$이므로
> 
> $$3c^2=7$$
> 
> $$c^2=\frac{7}{3}$$
> 
> $c>0$이므로
> 
> $$c=\sqrt{\frac{7}{3}}=\frac{\sqrt{7}}{\sqrt{3}}=\frac{\sqrt{21}}{3}$$

## 3. 평균값 정리의 응용

### 예제 169

모든 실수 $x$에 대하여 미분가능한 함수 $f(x)$가 $\lim_{x \to \infty}f'(x)=4$를 만족할 때 $\lim_{x \to \infty}\{f(x+2)-f(x-2)\}$의 값을 구하여라.

> [!summary]- 풀이
> 평균값 정리에 의해 $(x-2<c<x+2)$인 $c$가 존재하여
> 
> $$\frac{f(x+2)-f(x-2)}{(x+2)-(x-2)}=f'(c)$$
> 
> $$\frac{f(x+2)-f(x-2)}{4}=f'(c)$$
> 
> $x \to \infty$이면 $c \to \infty$이므로
> 
> $$\lim_{x \to \infty}\frac{f(x+2)-f(x-2)}{4}=\lim_{c \to \infty}f'(c)=4$$
> 
> 따라서
> 
> $$\lim_{x \to \infty}\{f(x+2)-f(x-2)\}=4 \times 4=16$$

### 예제 170

$\lim_{x \to \infty}(\cos\sqrt{x+1}-\cos\sqrt{x})$의 값을 구하여라.

> [!summary]- 풀이
> **※주의 33강 102쪽**
> 
> **"모든 실수 x에서 미분가능" 이 아니고 "x > 0에서 미분가능" 입니다.**
> 
> $f(x)=\cos\sqrt{x}$라 두면 $f(x)$는 $x>0$에서 미분가능하므로
> 
> 평균값 정리에 의해 $(x<c<x+1)$인 $c$가 존재하여
> 
> $$\frac{f(x+1)-f(x)}{(x+1)-x}=f'(c)$$
> 
> $$f(x+1)-f(x)=f'(c)$$
> 
> $$\cos\sqrt{x+1}-\cos\sqrt{x}=f'(c)$$
> 
> $x \to \infty$이면 $c \to \infty$이므로
> 
> $$\lim_{x \to \infty}(\cos\sqrt{x+1}-\cos\sqrt{x})=\lim_{c \to \infty}f'(c)$$
> 
> $f'(x)=-\frac{1}{2\sqrt{x}}\sin\sqrt{x}$이므로
> 
> $$\lim_{c \to \infty}f'(c)=\lim_{c \to \infty}\left(-\frac{1}{2\sqrt{c}}\sin\sqrt{c}\right)$$
> 
> $-1 \leq \sin\sqrt{c} \leq 1$이므로
> 
> $$-\frac{1}{2\sqrt{c}} \leq -\frac{1}{2\sqrt{c}}\sin\sqrt{c} \leq \frac{1}{2\sqrt{c}}$$
> 
> $\lim_{c \to \infty}-\frac{1}{2\sqrt{c}}=0$이고 $\lim_{c \to \infty}\frac{1}{2\sqrt{c}}=0$이므로
> 
> 샌드위치 정리에 의해
> 
> $$\lim_{c \to \infty}\left(-\frac{1}{2\sqrt{c}}\sin\sqrt{c}\right)=0$$
> 
> $$\therefore \lim_{x \to \infty}(\cos\sqrt{x+1}-\cos\sqrt{x})=0$$

### 예제 171

$\lim_{x \to 0^+}\frac{e^x-e^{\sin x}}{x-\sin x}$의 값을 평균값의 정리를 이용하여 구하시오.

> [!summary]- 풀이
> $f(x)=e^x$라 두면 $f(\sin x)=e^{\sin x}$
> 
> 평균값 정리에 의해 $(\sin x<c<x)$인 $c$가 존재하여
> 
> $$\frac{f(x)-f(\sin x)}{x-\sin x}=f'(c)$$
> 
> $x \to 0^+$이면 $\sin x \to 0^+$이고 $c \to 0^+$이므로
> 
> $$\lim_{x \to 0^+}\frac{e^x-e^{\sin x}}{x-\sin x}=\lim_{c \to 0^+}f'(c)$$
> 
> $f'(x)=e^x$이므로
> 
> $$\lim_{c \to 0^+}f'(c)=\lim_{c \to 0^+}e^c=e^0=1$$
> 
> $$\therefore \lim_{x \to 0^+}\frac{e^x-e^{\sin x}}{x-\sin x}=1$$

## 4. 접선의 방정식 (추가 정리)

### Thm (33): 접선의 방정식

**(1) 점선의 기울기** $\Rightarrow$ $y=f(x)$ 위의 점 $(a, f(a))$에서의 접선의 기울기는 이 점에서의 미분계수 즉, $f'(a)$이다.

**(2) 점선의 방정식** $\Rightarrow$ $y=f(a)(x-a)+f(a)$

**법선의 방정식** $\Rightarrow$ $y=-\frac{1}{f'(a)}(x-a)+f(a)$
