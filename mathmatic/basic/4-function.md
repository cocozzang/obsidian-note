# 7. 일, 이차함수

## 01 일차함수 (or 직선의 방정식)

### 1) 기울기 $m$, $y$절편 $n$인 직선의 방정식

$$y = mx + n$$

### 2) 기울기 $m$, 한점 $(x_1, y_1)$인 직선의 방정식

$$y - y_1 = m(x - x_1)$$

### 3) 기울기 $m = \tan\theta$

$\theta$는 직선과 $x$축의 양의 방향이 이루는 각

```
    y
    |  /
    | /
    |/_____ x
  θ
```

기울기: $m = \frac{\Delta y}{\Delta x}$

### 4) 두 점 $(x_1, y_1)$, $(x_2, y_2)$를 지나는 직선의 방정식

$$y - y_1 = \frac{y_2 - y_1}{x_2 - x_1}(x - x_1)$$

### 5) 한 점 $(x_1, y_1)$과 직선 $ax + by + c = 0$까지의 거리

$$d = \frac{|ax_1 + by_1 + c|}{\sqrt{a^2 + b^2}}$$

```
        d
       /|
      / |
     /  ⊥
    /____|
```

### 📝 예제

**예제 1**: $(-1, 3)$과 $(2, 5)$를 지나는 직선의 방정식은?

**풀이**:
$$y - 3 = \frac{5-3}{2-(-1)}(x-(-1))$$
$$y - 3 = \frac{2}{3}(x+1)$$
$$3y - 9 = 2x + 2$$
$$2x - 3y + 11 = 0$$

---

**예제 2**: $(1, 3)$에서 $2x - 3y + 11 = 0$까지의 거리를 구하라.

**풀이**:
$$d = \frac{|2(1) - 3(3) + 11|}{\sqrt{2^2 + (-3)^2}}$$
$$= \frac{|2 - 9 + 11|}{\sqrt{4 + 9}}$$
$$= \frac{|4|}{\sqrt{13}}$$
$$= \frac{4\sqrt{13}}{13}$$

## 02 이차함수

### 1. 이차함수의 그래프

#### (1) $y = ax^2$

- $a > 0$: 아래로 볼록 (∪)
- $a < 0$: 위로 볼록 (∩)

```
   y              y
   |              |___
   |  /\          |   \___
   | /  \         |       \
   |/____\        |________\x
 O        x     O
  (a>0)          (a<0)
```

#### (2) $y = a(x-m)^2 + n$

$y = ax^2$의 그래프를:

- $x$축방향으로 $m$만큼
- $y$축방향으로 $n$만큼 평행이동한 그래프

① 꼭지점: $(m, n)$
② 대칭축: $x = m$
③

- $a > 0$일 때: $(m, n)$은 최솟값 $= n$
- $a < 0$일 때: $(m, n)$은 최댓값 $= n$

```
       y
       |
    (m,n)
       ↑
   a>0 |  /\        최솟값 = n
       | /  \
       |/____\__ x
```

```
       y  ____
       | /    \
   a<0 |/      \    최댓값 = n
       ↓(m,n)
       |________ x
```

**중요**: $y = ax^2 + bx + c = a(x + \frac{b}{2a})^2 - \frac{b^2-4ac}{4a}$

**예제**: $y = 2x^2 - 4x + 5$

**풀이**:
$$y = 2(x^2 - 2x) + 5$$
$$= 2(x^2 - 2x + 1 - 1) + 5$$
$$= 2(x-1)^2 - 2 + 5$$
$$= 2(x-1)^2 + 3$$

- 꼭지점: $(1, 3)$
- 대칭축: $x = 1$
- 최솟값: $3$

---

### 2. 이차함수 그래프와 $x$축과의 교점

이차함수 $y = ax^2 + bx + c$의 그래프와 $x$축과의 교점의 $x$좌표는
이차방정식 $ax^2 + bx + c = 0$의 실근이다.

**판별식** $D = b^2 - 4ac$로 판단:

| $D > 0$                  | $D = 0$               | $D < 0$          |
| ------------------------ | --------------------- | ---------------- |
| 서로 다른 두 점에서 만남 | 한 점에서 만남 (접함) | 만나지 않음      |
| $x = \alpha$ or $\beta$  | $x = \alpha$ (중근)   | 근이 없다 (허근) |

```
   D>0              D=0              D<0
    |               |                |
   /|\             /|\              /|\
  / | \           / | \            /   \
 /  |  \         /__|__\          /     \
/___|___\       /   α   \        /_______\
α       β
```

---

### (2) 이차함수의 그래프와 직선의 위치관계

$$
\begin{cases}
y = ax^2 + bx + c & (a > 0) \\
y = mx + n & (m > 0)
\end{cases}
$$

두 식을 연립해서 나온 이차방정식
$$ax^2 + (b-m)x + c - n = 0$$
의 판별식을 $D$라 하면:

① 서로 다른 두 점에서 만날 때: $D > 0$
② 한 점에서 만날 때(접할 때): $D = 0$
③ 만나지 않을 때: $D < 0$

```
       (1)
      /|(2)
     / |(3)
    /  |
   /   |
  /____|_____
```

---

### 3. 이차함수와 이차부등식

이차함수 $y = ax^2 + bx + c$ $(a > 0)$와 이차부등식의 해 사이에는 다음과 같은 관계가 있다.

|                                  | $D > 0$                           | $D = 0$                | $D < 0$    |
| -------------------------------- | --------------------------------- | ---------------------- | ---------- |
| $y = ax^2 + bx + c$의 그래프 $(a>0)$ | ∪형 (α, β 교점)                      | ∪형 (α 접점)              | ∪형 (교점 없음) |
| $ax^2 + bx + c > 0$의 해           | $x < \alpha$ 또는 $x > \beta$       | $x \neq \alpha$인 모든 실수 | 모든 실수      |
| $ax^2 + bx + c < 0$의 해           | $\alpha < x < \beta$              | 해가 없다                  | 해가 없다      |
| $ax^2 + bx + c \geq 0$의 해        | $x \leq \alpha$ 또는 $x \geq \beta$ | 모든 실수                  | 모든 실수      |
| $ax^2 + bx + c \leq 0$의 해        | $\alpha \leq x \leq \beta$        | $x = \alpha$           | 해가 없다      |

### 📝 예제

**예제 1**: $y = x^2 - 2x - 15$

**풀이**:
$$y = x^2 - 2x - 15 = (x-5)(x+3)$$

- 두 근: $x = -3, 5$
- 꼭지점: $x = \frac{-3+5}{2} = 1$
- $y = (1-5)(1+3) = -16$
- 꼭지점: $(1, -16)$

```
    y
    |     /
    |    / \
____|___/___\____x
   -3   1   5
        |
       (1,-16)
```

---

**예제 2**: $y = x^2 - 2x + 3$

**풀이**:
$$y = (x-1)^2 + 2$$

판별식: $D = (-2)^2 - 4(1)(3) = 4 - 12 = -8 < 0$

- $x$축과 만나지 않음
- 최솟값: $2$ (at $x=1$)

```
    y
    |
    |  /\
    | /  \
    |/____\
    1
  (1,2)
```

# 8. 유리, 무리함수

## 01 유리함수

### (1) 기본형 $y = \frac{k}{x}$ $(k \neq 0)$의 그래프

```
 y=k/x (k>0)        y=k/x (k<0)
    y                  y
    |  \               | /
    |   \              |/
    |____\__x        __|___x
  O      \          /  O
          \        /
           \      |
```

① 정의역: $x \neq 0$인 모든 실수
치역: $y \neq 0$인 모든 실수

② 점근선: $x = 0$ ($y$축), $y = 0$ ($x$축)

③ $k > 0$: 1, 3사분면
$k < 0$: 2, 4사분면

④ 원점에 대칭인 직각쌍곡선
$y = \pm x$에 대칭 (특히, 점근선의 교점을 지남)

---

### (2) $y = \frac{k}{x-p} + q$ $(k \neq 0)$의 그래프

$$y = \frac{k}{x} \xrightarrow{\substack{x축: p이동 \\ y축: q이동}} y - q = \frac{k}{x-p}$$
$$\Downarrow$$
$$y = \frac{k}{x-p} + q$$

① $y = \frac{k}{x}$의 그래프를 $x$축으로 $p$만큼, $y$축으로 $q$만큼 평행이동한 그래프

② 점근선: $x = p$, $y = q$

③ 정의역: $x \neq p$인 모든 실수
치역: $y \neq q$인 모든 실수

④ $(p, q)$에 대칭
$y = \pm(x-p) + q$에 대칭 (기울기 $\pm 1$, $(p, q)$를 지나는 직선)

```
    y
    |
  q |........
    |     p
  O |_____|___ x
    p
```

### 📝 예제

**예제 1**: $y = \frac{1}{x-2} + 3$

- 점근선: $x = 2$, $y = 3$
- 대칭점: $(2, 3)$
- 정의역: $x \neq 2$
- 치역: $y \neq 3$

---

**예제 2**: $y = \frac{2x-3}{x+1}$

**풀이**: 분자를 분모로 나누는 형태로 변환
$$y = \frac{2x-3}{x+1} = \frac{2(x+1) - 5}{x+1} = \frac{2(x+1)}{x+1} - \frac{5}{x+1}$$
$$= 2 - \frac{5}{x+1}$$
$$= \frac{-5}{x+1} + 2$$

- 점근선: $x = -1$, $y = 2$
- 대칭점: $(-1, 2)$

---

### (3) 역함수 구하기

$$y = \frac{ax+b}{cx+d} \Rightarrow x = \frac{ay+b}{cy+d}$$

$y$에 대해 정리하면:
$$x(cy+d) = ay+b$$
$$cxy + dx = ay + b$$
$$cxy - ay = b - dx$$
$$y(cx-a) = b - dx$$
$$y = \frac{-dx+b}{cx-a} = \frac{dx-b}{-cx+a}$$

따라서 $f^{-1}(x) = \frac{dx-b}{-cx+a}$

**예제**: $y = \frac{2x-3}{x+1}$의 역함수는?

**풀이**:

- $a=2, b=-3, c=1, d=1$
- $f^{-1}(x) = \frac{1 \cdot x - (-3)}{-1 \cdot x + 2} = \frac{x+3}{-x+2} = \frac{x+3}{2-x}$

또는 $y = \frac{2x-3}{x+1}$에서 $x$와 $y$를 바꾸면:
$$x = \frac{2y-3}{y+1}$$
$$x(y+1) = 2y-3$$
$$xy + x = 2y - 3$$
$$xy - 2y = -x - 3$$
$$y(x-2) = -x-3$$
$$y = \frac{-x-3}{x-2} = \frac{x+3}{2-x}$$

**참고**: 행렬 $A = \begin{pmatrix} a & b \\ c & d \end{pmatrix}$의 역행렬
$$A^{-1} = \frac{1}{ad-bc}\begin{pmatrix} d & -b \\ -c & a \end{pmatrix}$$

유리함수: $y = \frac{ax+b}{cx+d}$
역함수: $y = \frac{dx-b}{-cx+a}$

## 02 무리함수

### (1) $y = p\sqrt{ax}$의 그래프

$y = \sqrt{x}$ $(p > 0, a > 0)$

```
    y
    |  ___
    | /
    |/
    |_______x
  O
```

① 꼭지점: $(0, 0)$

② 정의역: $x \geq 0$ (근호안 $\geq 0$)
치역: $y \geq 0$ ($y$의 양의 제곱근)

**부호에 따른 그래프**:

```
            y
            |
    y=√(-x) |___  ___y=√x
            |
  x≤0, y≥0  |  x≥0, y≥0
____________|____________x
            O
  x≤0, y≤0  |  x≥0, y≤0
            |
   y=-√(-x)_|____y=-√x
            |
```

---

### (2) $y - n = p\sqrt{a(x-m)}$

$$y = p\sqrt{ax} \xrightarrow{\text{평행이동}} y - n = p\sqrt{a(x-m)}$$
$$\text{x축: }m$$
$$\text{y축: }n$$
$$\Downarrow$$
$$y = p\sqrt{a(x-m)} + n$$

```
    y
    |
  n |___
    |
  O |___m___ x
    m
```

### 📝 예제

**예제 1**: $y = 2\sqrt{x}$와 $y = 2\sqrt{x-1} - 3$를 그려라.

**풀이**:

1. $y = 2\sqrt{x}$: 꼭지점 $(0, 0)$, 정의역 $x \geq 0$

2. $y = 2\sqrt{x-1} - 3$: $y = 2\sqrt{x}$를 $x$축으로 $1$, $y$축으로 $-3$ 평행이동
   - 꼭지점: $(1, -3)$
   - 정의역: $x \geq 1$

---

**예제 2**: $y = \sqrt{-x+3} - 1$를 그려라.

**풀이**:
$$y = \sqrt{-(x-3)} - 1$$

- $y = \sqrt{-x}$를 $x$축으로 $3$, $y$축으로 $-1$ 평행이동
- 꼭지점: $(3, -1)$
- 정의역: $x \leq 3$ (근호 안 $-x+3 \geq 0$)

## 03 절댓값 기호를 포함한 함수

### 1. $|x| = \begin{cases} x & (x \geq 0) \\ -x & (x < 0) \end{cases}$

#### (1) $y = |f(x)|$: $x$축 아랫부분을 접어올린다

#### (2) $y = f(|x|)$: $x \geq 0$인 부분을 $y$축에 대칭

#### (3) $|y| = f(x)$: $y \geq 0$인 부분을 $x$축에 대칭

#### (4) $|y| = f(|x|)$: $x \geq 0$, $y \geq 0$인 부분을 $x$축, $y$축, 원점 대칭

### 📝 예제

    |    /\
    |   /  \
____|__/__\___ x
  O  1    2
```

**(1) $y = |x^2 - 3x + 2|$**

_/__|______\_ x
 O  1   2
```

**(2) $y = x^2 - 3|x| + 2$**

$x \geq 0$ 부분을 $y$축 대칭

```

_/__\_\/__\___ x

**(3) $|y| = x^2 - 3x + 2$**

    |  /\

```

**(4) $|y| = x^2 - 3|x| + 2$**

1사분면 부분을 모든 사분면으로 대칭

### 우함수, 기함수

**우함수**: $f(-x) = f(x)$ - 그래프가 $y$축에 대하여 대칭

**기함수**: $f(-x) = -f(x)$ - 그래프가 원점에 대하여 대칭

**예제**:

- $y = x^2$: $f(-x) = (-x)^2 = x^2 = f(x)$ → 우함수
- $y = x^3$: $f(-x) = (-x)^3 = -x^3 = -f(x)$ → 기함수

---

**예제**: $f(x)$가 우함수, $g(x)$가 기함수일 때 다음 함수들이 우함수인지, 기함수인지 판단하라.

① $f(x) - g(x)$
② $f(x) \cdot g(x)$
③ $(f \circ g)(x)$

**풀이**:
① $h(-x) = f(-x) - g(-x) = f(x) - (-g(x)) = f(x) + g(x)$
$\neq h(x)$, $\neq -h(x)$ → 우함수도 기함수도 아님

② $h(-x) = f(-x) \cdot g(-x) = f(x) \cdot (-g(x)) = -f(x)g(x) = -h(x)$ → 기함수

③ $(f \circ g)(-x) = f(g(-x)) = f(-g(x))$
$f$가 우함수이므로 $f(-g(x)) = f(g(x)) = (f \circ g)(x)$ → 우함수

## 🔑 핵심 정리

### 일차함수

- 기울기 $m$, $y$절편 $n$: $y = mx + n$
- 두 점을 지나는 직선: 기울기를 구한 후 공식 적용
- 점과 직선 사이의 거리 공식 암기

### 이차함수

- 표준형: $y = a(x-m)^2 + n$ (꼭지점 $(m, n)$)
- 판별식으로 $x$축과의 교점 개수 판단
- 이차부등식의 해는 그래프로 직관적으로 이해

### 유리함수

- 기본형 $y = \frac{k}{x}$: 쌍곡선
- 점근선을 먼저 파악하고 그래프 그리기
- 역함수는 계수를 바꾸는 규칙 활용

### 무리함수

- 근호 안이 0 이상이어야 함 (정의역 주의)
- 제곱근의 양수 조건 (치역 주의)

### 절댓값 함수

- $|f(x)|$: 음수 부분을 위로
- $f(|x|)$: 양수 부분을 좌우 대칭
- 경계점을 찾아 구간별로 나누어 분석

### 가우스 함수

- 최대정수함수: 계단 모양
- $x - [x]$: 톱니 모양 (주기 1)

## 📚 연습문제

### 문제 1

$(x-3)^2 + (y-1)^2 = 2$를 $y = x$에 대하여 대칭이동한 원의 중심이 $y = ax + 5$ 위에 있을 때 상수 $a$의 값은?

**풀이**:
$y = x$에 대칭 → $(x, y) \leftrightarrow (y, x)$
원의 중심 $(3, 1)$이 $(1, 3)$으로 이동
$(1, 3)$이 $y = ax + 5$ 위에 있으므로:
$$3 = a(1) + 5$$
$$a = -2$$

### 문제 3

집합 $X = \{1, 2, 3, 4, 5\}$라 하자. 함수 $f: X \to X$는

$$
f(x) = \begin{cases}
x + 3 & (x = 1, 2) \\
x - 2 & (x = 3, 4, 5)
\end{cases}
$$

와 같이 정의한다. 함수 $g: X \to X$가 $g(1) = 2$이고 $f \circ g = g \circ f$를 만족하도록 $g$를 결정하여라.

**힌트**: $f \circ g = g \circ f$를 만족하려면 각 원소에 대해 합성 순서가 같아야 함
