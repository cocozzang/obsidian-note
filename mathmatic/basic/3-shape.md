## 5. 도형의 이동

### 도형 관련 기본공식

① **두 점 사이의 거리**: $(x_1, y_1)$, $(x_2, y_2)$
$$d = \sqrt{(x_1-x_2)^2 + (y_1-y_2)^2}$$

② **직선의 방정식**: $y = ax + b$

③ **원의 방정식**: 중심이 $(a, b)$이고 반지름 $r$인 원
$$(x-a)^2 + (y-b)^2 = r^2$$

---

### 01 평행이동

#### (1) 점의 이동

$$f: (x, y) \to (x+m, y+n)$$

점 $(x, y)$를 $\begin{cases} x\text{축으로 } m\text{만큼} \\ y\text{축으로 } n\text{만큼} \end{cases}$ 평행이동

#### (2) 도형의 이동

$$f(x, y) = 0 \xrightarrow{\begin{matrix} x\text{축: } m \\ y\text{축: } n \end{matrix}} f(x-m, y-n) = 0$$

**예제**: $y = 2x + 1$을 $x$축으로 3만큼, $y$축으로 2만큼 평행이동한 식은?

**풀이**:
$$x \to x-3, \quad y \to y-2$$
$$y-2 = 2(x-3) + 1$$
$$y-2 = 2x-6+1$$
$$y = 2x-3$$

### 02 대칭이동

#### (1) 좌표축에 대한 대칭이동

| 점 $(x, y)$        | 도형 $f(x, y) = 0$ |
| ------------------ | ------------------ |
| ① $x$축: $(x, -y)$ | $f(x, -y) = 0$     |
| ② $y$축: $(-x, y)$ | $f(-x, y) = 0$     |
| ③ 원점: $(-x, -y)$ | $f(-x, -y) = 0$    |

#### (2) 직선에 대한 대칭이동

| 점 $(x, y)$                   | 도형 $f(x, y) = 0$  |
| ----------------------------- | ------------------- |
| ① $y = x$: $(y, x)$           | $f(y, x) = 0$       |
| ② $y = -x$: $(-y, -x)$        | $f(-y, -x) = 0$     |
| ③ $x = a$: $(2a-x, y)$        | $f(2a-x, y) = 0$    |
| ④ $y = b$: $(x, 2b-y)$        | $f(x, 2b-y) = 0$    |
| ⑤ 점 $(a, b)$: $(2a-x, 2b-y)$ | $f(2a-x, 2b-y) = 0$ |

**증명 예시** ($y = x$에 대한 대칭):

점 $(x, y)$를 $y = x$에 대해 대칭이동한 점을 $(x', y')$라 하면:

- 두 점을 잇는 직선의 기울기 $\times$ $y = x$의 기울기 = -1
- 두 점의 중점이 $y = x$ 위에 있음

이 조건들을 풀면 $x' = y, y' = x$

**예제**: $2x - y + 5 = 0$을 $x$축에 대하여 대칭이동한 후 $x = 1$에 대하여 대칭이동한 그래프는?

**풀이**:

1. $x$축 대칭: $y \to -y$
   $$2x - (-y) + 5 = 0 \Rightarrow 2x + y + 5 = 0$$

2. $x = 1$ 대칭: $x \to 2-x$
   $$2(2-x) + y + 5 = 0$$
   $$4-2x+y+5 = 0$$
   $$-2x+y+9 = 0$$
   $$2x-y-9 = 0$$

## 6. 함수와 그래프

### 01 함수의 정의

#### (1) 함수

공집합이 아닌 두 집합 $X$, $Y$에 대하여 $X$의 각 원소에 $Y$의 각 원소가 하나씩 빠짐없이 대응될 때 $X$에서 $Y$로의 함수라 하고 이것을 기호로 $f: X \to Y$로 나타낸다.

- $X$: 정의역 (domain)
- $Y$: 공역 (codomain)
- $f(x)$: 치역 (range), $f(x) \subset Y$

#### (2) 상등 (서로 같은 함수) $f = g$

두 함수 $f$, $g$에 대하여 $f$와 $g$의 다음이 모두 같을 때 $f = g$라 한다:

- ① 정의역
- ② 공역
- ③ 함숫값

**예제 1**: 정의역 $X = \{-1, 1\}$에서 $f(x) = x$, $g(x) = x^3$. 두 함수는 같은가?

**풀이**:

- $f(-1) = -1, f(1) = 1$
- $g(-1) = -1, g(1) = 1$
- 정의역, 공역, 함숫값이 모두 같으므로 $f = g$ ✓

**예제 2**: 정의역 $X = \{0, 1\}$에서 $f(x) = x$, 정의역 $X' = \{2, 3\}$에서 $g(x) = x$이다. 두 함수는 같은가?

**풀이**: 정의역이 다르므로 $f \neq g$ ✗

---

### 02 함수의 종류

#### (1) 일대일 함수 (Injective Function)

함수 $f: X \to Y$에서 $X$의 임의의 두 원소 $x_1, x_2$에 대해서
$$x_1 \neq x_2 \Rightarrow f(x_1) \neq f(x_2)$$

**대우**: $f(x_1) = f(x_2) \Rightarrow x_1 = x_2$

#### (2) 일대일 대응 (※역함수 존재조건)

① $x_1 \neq x_2 \Rightarrow f(x_1) \neq f(x_2)$
② 치역 = 공역

**예제 1**: 일차함수 $y = ax + b$ (직선)

- 일대일 대응 ✓ (항상 쭉 증가 또는 쭉 감소)

**예제 2**: 이차함수 $y = ax^2 + bx + c$ (포물선) ∪ 또는 ∩

- 일대일 대응 ✗ (정의역을 제한하면 가능)

**예제 3**: 지수함수 $y = a^x$

- 일대일 대응 ✓

⇨ **일대일이 되려면 그래프 모양이 '쭉 증가' 또는 '쭉 감소'여야 한다.**

#### (3) 항등함수 (Identity Function)

함수 $f: X \to X$일 때 $f(x) = x$일 때 $f$를 항등함수라고 한다.

모든 항등함수는 일대일 대응이다.

#### (4) 상수함수 (Constant Function)

함수 $f: X \to c$ (상수, 하나의 값), $f(x) = c$

---

### 03 합성함수

두 함수 $f: X \to Y$, $g: Y \to Z$가 주어져 있을 때 $X$의 원소 $x$에 대하여 $Z$의 원소 $g(f(x))$를 대응시킴으로써 $X$를 정의역, $Z$를 공역으로 하는 새로운 함수를 $f$와 $g$의 합성함수라 하고 $g \circ f$로 나타낸다.

$$g \circ f: X \to Z, \quad (g \circ f)(x) = g(f(x)) \quad (x \in X)$$

```
X ──f──> Y ──g──> Z
 \                 /
  \___g ∘ f______/
```

#### 합성함수의 성질

① $g \circ f \neq f \circ g$ (교환법칙이 성립하지 않는다.)

② $(h \circ g) \circ f = h \circ (g \circ f)$ (결합법칙 성립)

③ $f: X \to X$, $f \circ I = I \circ f = f$ ($I$는 $X$에서의 항등함수)

**예제**: $f(x) = 2x + 1$, $g(x) = x^2$일 때 다음을 구하시오.

1. $(g \circ f)(x)$
   $$g(f(x)) = g(2x+1) = (2x+1)^2 = 4x^2 + 4x + 1$$

2. $(f \circ g)(x)$
   $$f(g(x)) = f(x^2) = 2x^2 + 1$$

3. $g \circ f \neq f \circ g$ ✓

### 04 역함수

#### (1) 역함수의 정의

함수 $f: X \to Y$가 일대일 대응일 때, $Y$의 임의의 한 원소에 $X$의 원소가 하나씩만 대응이 되고 이것은 $Y$에서 $X$로의 함수가 된다.

이 새로운 함수를 함수 $f$의 역함수라 하고 $f^{-1}: Y \to X$로 나타낸다.

① 역함수: $f^{-1}: Y \to X$
② $f(x) = y \Leftrightarrow f^{-1}(y) = x$

**※ 일대일 대응이어야만 하는 이유**:

- 일대일이 아니면: $Y$의 한 원소에 $X$의 여러 원소가 대응 → 함수 아님
- 치역 ≠ 공역이면: $Y$의 일부 원소에 대응하는 $X$의 원소가 없음 → 함수 아님

#### (2) 역함수의 성질

① $f \circ f^{-1} = f^{-1} \circ f = I$ (항등함수)
$$\Rightarrow f \circ g = I \text{이면 } g = f^{-1}, f = g^{-1}$$

② $(f^{-1})^{-1} = f$

③ $(g \circ f)^{-1} = f^{-1} \circ g^{-1}$
$$(h \circ g \circ f)^{-1} = f^{-1} \circ g^{-1} \circ h^{-1}$$

④ $f^{-1}(a) = b \Leftrightarrow f(b) = a$ (역함수는 $x$, $y$가 뒤집어진다)

#### (3) 역함수 구하기

① 함수 $y = f(x)$가 일대일 대응인가를 확인한다.
② $y = f(x)$에서 $x$와 $y$를 바꾼다. 즉, $x = f(y)$
③ $x = f(y)$를 $y$에 대해 정리하면 $y = f^{-1}(x)$이다.

**예제 1**: $y = \frac{1}{2}x + 1$의 역함수를 구하여라.

**풀이**:
$$x = \frac{1}{2}y + 1$$
$$\frac{1}{2}y = x - 1$$
$$y = 2x - 2$$
$$\therefore f^{-1}(x) = 2x - 2$$

**예제 2**: $y = x^2 - 3$ ($x \geq 0$, $y \geq -3$)의 역함수를 구하여라.

**풀이**:
$$x = y^2 - 3 \quad (y \geq 0)$$
$$y^2 = x + 3$$
$$y = \sqrt{x+3} \quad (y \geq 0)$$
$$\therefore f^{-1}(x) = \sqrt{x+3}$$

#### (4) 역함수의 그래프 ($x$와 $y$를 뒤집는다)

함수 $y = f(x)$의 그래프와 그 역함수 $y = f^{-1}(x)$의 그래프는 직선 $y = x$에 대하여 대칭이다.

$$(x, y) \xleftrightarrow{y=x\text{에 대칭}} (y, x)$$

**이유**:

- $f$: $(a, b)$ → $y = f(x)$에서 $x = a$일 때 $y = b$
- $f^{-1}$: $(b, a)$ → $y = f^{-1}(x)$에서 $x = b$일 때 $y = a$
- 두 점 $(a, b)$와 $(b, a)$는 $y = x$에 대해 대칭

## 정리

### 도형의 이동

1. **평행이동**: $x \to x-m$, $y \to y-n$
2. **대칭이동**:
   - 좌표축: $x$축 $(x, -y)$, $y$축 $(-x, y)$, 원점 $(-x, -y)$
   - 직선: $y = x$ → $(y, x)$, $y = -x$ → $(-y, -x)$

### 함수의 성질

1. **일대일 함수**: $x_1 \neq x_2 \Rightarrow f(x_1) \neq f(x_2)$
2. **일대일 대응**: 일대일 + 치역 = 공역
3. **합성함수**: $(g \circ f)(x) = g(f(x))$
4. **역함수**: $f^{-1}$는 $y = x$에 대칭, 일대일 대응일 때만 존재

### 역함수 구하는 순서

1. 일대일 대응 확인
2. $x$와 $y$ 교환
3. $y$에 대해 정리
4. $y = x$에 대칭 확인
