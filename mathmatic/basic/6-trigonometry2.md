# 10. 삼각함수(2)

## 01 삼각함수의 그래프

### (1) $y = \sin x$

```
      y
    1 |    ╱╲    ╱╲    ╱╲
      |   ╱  ╲  ╱  ╲  ╱  ╲
      |  ╱    ╲╱    ╲╱    ╲
  ────┼──────────────────────── x
    O | π/2  π  3π/2 2π
   -1 |
```

#### 성질

① **정의역**: 모든 실수 ($\mathbb{R}$)

② **치역**: $-1 \leq y \leq 1$
   - 최댓값: $1$ (at $x = \frac{\pi}{2} + 2\pi n$)
   - 최솟값: $-1$ (at $x = \frac{3\pi}{2} + 2\pi n$)

③ **주기**: $2\pi$
   $$\sin(x + 2\pi) = \sin x$$

④ **대칭성**: 원점대칭 (기함수)
   $$\sin(-x) = -\sin x$$
   $$f(-x) = -f(x)$$

⑤ **영점** ($y = 0$인 점): $x = n\pi$ ($n$은 정수)

---

### (2) $y = \cos x$

```
      y
    1 |╲  ╱╲  ╱╲  ╱
      | ╲╱  ╲╱  ╲╱
      |            
  ────┼──────────────────────── x
    O | π/2  π  3π/2 2π
   -1 |
```

#### 성질

① **정의역**: 모든 실수 ($\mathbb{R}$)

② **치역**: $-1 \leq y \leq 1$
   - 최댓값: $1$ (at $x = 2\pi n$)
   - 최솟값: $-1$ (at $x = \pi + 2\pi n$)

③ **주기**: $2\pi$
   $$\cos(x + 2\pi) = \cos x$$

④ **대칭성**: $y$축 대칭 (우함수)
   $$\cos(-x) = \cos x$$
   $$f(-x) = f(x)$$

⑤ **영점**: $x = \frac{\pi}{2} + n\pi$ ($n$은 정수)

---

### (3) $y = \tan x$

```
      y
    3 |  |     |     |
    2 |  |   ╱ |   ╱ |
    1 | ╱|  ╱  |  ╱  |
  ────┼──┼─────┼─────┼──── x
   -π/2| O  π/2  π  3π/2
   -1 | |     |     |
   -2 | |     |     |
```

#### 성질

① **정의역**: $x \neq \frac{\pi}{2} + n\pi$ ($n$은 정수)
   - $\cos x = 0$인 점에서 정의되지 않음

② **치역**: 모든 실수 ($\mathbb{R}$)
   - 최댓값, 최솟값 없음

③ **주기**: $\pi$
   $$\tan(x + \pi) = \tan x$$
   - $\sin$, $\cos$보다 주기가 절반!

④ **대칭성**: 원점대칭 (기함수)
   $$\tan(-x) = -\tan x$$

⑤ **점근선**: $x = \frac{\pi}{2} + n\pi$ ($n$은 정수)

⑥ **영점**: $x = n\pi$ ($n$은 정수)

---

### (4) 삼각함수 그래프 비교

| 함수 | 주기 | 대칭 | 치역 | 영점 |
|------|------|------|------|------|
| $\sin x$ | $2\pi$ | 원점대칭 | $[-1, 1]$ | $n\pi$ |
| $\cos x$ | $2\pi$ | $y$축대칭 | $[-1, 1]$ | $\frac{\pi}{2} + n\pi$ |
| $\tan x$ | $\pi$ | 원점대칭 | $\mathbb{R}$ | $n\pi$ |

#### 성질
- **주기**: $\pi$ (원래의 절반!)
- **치역**: $[0, 1]$
- **영점**: $x = n\pi$

**변환**: $x$축 아랫부분을 위로 접어올림

      y

#### 성질
- **주기**: $\pi$
- **치역**: $[0, 1]$
- **영점**: $x = \frac{\pi}{2} + n\pi$

---

### (3) $y = |\tan x|$

**변환**: $x$축 아랫부분을 위로 접어올림

      y
      | |  |  |  |

#### 성질
- **주기**: $\frac{\pi}{2}$ (원래의 절반!)
- **치역**: $[0, \infty)$
- **영점**: $x = n\pi$
- **점근선**: $x = \frac{\pi}{2} + n\pi$

## 03 여러 가지 삼각함수 그래프

### (1) $y = 2\sin x$

**변환**: 진폭이 2배

```
      y
    2 |    ╱╲    
    1 |   ╱  ╲   
      |  ╱    ╲  
  ────┼──────────────── x
    O | π/2  π  3π/2
   -1 |     ╲  ╱
   -2 |      ╲╱
```

- **진폭**: $2$ (최댓값 $2$, 최솟값 $-2$)
- **주기**: $2\pi$ (변화 없음)

---

### (2) $y = \sin(-x)$

**변환**: $y$축 대칭 = $-\sin x$

```
      y
    1 |╲    ╱╲    ╱
      | ╲  ╱  ╲  ╱
      |  ╲╱    ╲╱
  ────┼──────────────── x
      | π/2  π  3π/2
   -1 |
```

$$y = \sin(-x) = -\sin x$$

---

### (3) $y = -\sin x$

**변환**: $x$축 대칭 (상하 반전)

```
      y
    1 |
      |  ╲╱    ╲╱
      | ╱  ╲  ╱  ╲
  ────┼──────────────── x
      | π/2  π  3π/2
   -1 |╱    ╲╱    ╲
```

---

### (4) $y = \sin 2x$

**변환**: 주기가 절반

```
      y
    1 |  ╱╲╱╲╱╲╱╲
      | ╱        ╲
      |╱          ╲
  ────┼──────────────── x
    O | π/2  π  3π/2
   -1 |
```

- **주기**: $\frac{2\pi}{2} = \pi$
- **일반형**: $y = \sin(nx)$의 주기는 $\frac{2\pi}{n}$

---

### (5) $y = \sin\frac{x}{2}$

**변환**: 주기가 2배

```
      y
    1 |      ╱╲    
      |     ╱  ╲   
      |    ╱    ╲  
  ────┼────────────────────── x
    O |  π  2π  3π  4π
   -1 |         ╲  ╱
      |          ╲╱
```

- **주기**: $\frac{2\pi}{1/2} = 4\pi$

---

### (6) $y = \sin\left(x - \frac{\pi}{2}\right)$

**변환**: $x$축 방향으로 $\frac{\pi}{2}$ 평행이동 (오른쪽)

```
      y
    1 |      ╱╲    ╱╲
      |     ╱  ╲  ╱  
      |    ╱    ╲╱   
  ────┼──────────────────── x
    O | π/2  π  3π/2 2π
   -1 |
```

$$y = \sin\left(x - \frac{\pi}{2}\right) = -\cos x$$

---

### (7) $y = \sin x + 3$

**변환**: $y$축 방향으로 $3$ 평행이동 (위로)

```
      y
    4 |    ╱╲    
    3 |   ╱  ╲───
    2 |  ╱    ╲  
  ────┼──────────────── x
    O | π/2  π  3π/2
```

- **최댓값**: $4$
- **최솟값**: $2$
- **중심선**: $y = 3$

---

### (8) $y = 3\sin(2x - 4) + 1$

**일반형**: $y = A\sin(Bx + C) + D$

**분석**:
$$y = 3\sin\left(2\left(x - 2\right)\right) + 1$$

- **진폭** $|A|$: $3$
- **주기** $\frac{2\pi}{|B|}$: $\frac{2\pi}{2} = \pi$
- **위상** (평행이동): $x$축으로 $2$ (오른쪽)
- **수직이동** $D$: $y$축으로 $1$ (위로)
- **최댓값**: $3 + 1 = 4$
- **최솟값**: $-3 + 1 = -2$

## 04 삼각함수 그래프 정리

### 일반형: $y = A\sin(Bx + C) + D$

| 요소 | 값 | 의미 |
|------|------|------|
| $A$ | 진폭 | 최댓값과 최솟값의 차이의 절반 |
| $B$ | 주기 결정 | 주기 = $\frac{2\pi}{\|B\|}$ |
| $C$ | 위상 | $x$축 평행이동 = $-\frac{C}{B}$ |
| $D$ | 수직이동 | $y$축 평행이동 |

### 변환 정리

| 그래프 | 주기 | 진폭 | 설명 |
|--------|------|------|------|
| $y = \sin x$ | $2\pi$ | $1$ | 기본형 |
| $y = A\sin x$ | $2\pi$ | $\|A\|$ | 진폭 변화 |
| $y = \sin(Bx)$ | $\frac{2\pi}{\|B\|}$ | $1$ | 주기 변화 |
| $y = \sin(x - C)$ | $2\pi$ | $1$ | 수평이동 |
| $y = \sin x + D$ | $2\pi$ | $1$ | 수직이동 |
| $y = \|\sin x\|$ | $\pi$ | $1$ | 절댓값 (주기 절반) |

## 11. 삼각방정식과 부등식

### (1) 삼각방정식

#### ① 기본형: $\sin x = a$ ($0 \leq x < 2\pi$)

**예제**: $\sin x = \frac{1}{2}$ ($0 \leq x < 2\pi$)

**풀이**:

방법1 (단위원 이용):
```
      y
      |
   1/2|___
      |  /|\
      | / | \
  ────┼────────── x
    O |
```

$\sin x = \frac{1}{2}$인 각도:
- 1분면: $x = \frac{\pi}{6}$
- 2분면: $x = \pi - \frac{\pi}{6} = \frac{5\pi}{6}$

$$\therefore x = \frac{\pi}{6}, \frac{5\pi}{6}$$

방법2 (계열각):
- $\sin x = \frac{1}{2}$인 기본각: $\frac{\pi}{6}$
- $\frac{\pi}{6}$ 계열: $\frac{\pi}{6}, \frac{5\pi}{6}, \frac{13\pi}{6}, ...$
- $0 \leq x < 2\pi$ 범위: $x = \frac{\pi}{6}, \frac{5\pi}{6}$

---

#### ② 기본형: $\cos x = a$ ($0 \leq x < 2\pi$)

**예제**: $\cos x = -\frac{1}{2}$ ($0 \leq x < 2\pi$)

**풀이**:

$\cos x = -\frac{1}{2}$인 각도:
- 2분면: $x = \pi - \frac{\pi}{3} = \frac{2\pi}{3}$
- 3분면: $x = \pi + \frac{\pi}{3} = \frac{4\pi}{3}$

$$\therefore x = \frac{2\pi}{3}, \frac{4\pi}{3}$$

---

#### ③ 기본형: $\tan x = a$ ($0 \leq x < \pi$)

**예제**: $\tan x = 1$ ($0 \leq x < \pi$)

**풀이**:

$\tan x = 1$인 기본각: $\frac{\pi}{4}$

$\tan$의 주기가 $\pi$이므로 $0 \leq x < \pi$ 범위에서:

$$x = \frac{\pi}{4}$$

만약 $0 \leq x < 2\pi$라면: $x = \frac{\pi}{4}, \frac{5\pi}{4}$

---

### 📝 삼각방정식 예제

**예제 1**: $2\sin^2 x - \sin x - 1 = 0$ ($0 \leq x < 2\pi$)

**풀이**:

$\sin x = t$로 치환:
$$2t^2 - t - 1 = 0$$
$$(2t + 1)(t - 1) = 0$$
$$t = -\frac{1}{2} \text{ 또는 } t = 1$$

① $\sin x = -\frac{1}{2}$
- 3분면: $x = \pi + \frac{\pi}{6} = \frac{7\pi}{6}$
- 4분면: $x = 2\pi - \frac{\pi}{6} = \frac{11\pi}{6}$

② $\sin x = 1$
- $x = \frac{\pi}{2}$

$$\therefore x = \frac{\pi}{2}, \frac{7\pi}{6}, \frac{11\pi}{6}$$

---

**예제 2**: $\cos 2x = \frac{1}{2}$ ($0 \leq x < 2\pi$)

**풀이**:

$2x = t$로 치환하면 $0 \leq t < 4\pi$

$\cos t = \frac{1}{2}$의 해:
- $t = \frac{\pi}{3}, \frac{5\pi}{3}, \frac{7\pi}{3}, \frac{11\pi}{3}$

$2x = t$이므로:
$$x = \frac{\pi}{6}, \frac{5\pi}{6}, \frac{7\pi}{6}, \frac{11\pi}{6}$$

---

**예제 3**: $\tan x = -\sqrt{3}$ ($0 \leq x < 2\pi$)

**풀이**:

$\tan x = -\sqrt{3}$의 기본각: $\tan\frac{\pi}{3} = \sqrt{3}$

음수이므로 2분면 또는 4분면:
- 2분면: $x = \pi - \frac{\pi}{3} = \frac{2\pi}{3}$
- 4분면: $x = 2\pi - \frac{\pi}{3} = \frac{5\pi}{3}$

$$\therefore x = \frac{2\pi}{3}, \frac{5\pi}{3}$$

---

### (2) 삼각부등식

#### ① $\sin x > a$ 또는 $\sin x < a$

**예제**: $\sin x > \frac{1}{2}$ ($0 \leq x < 2\pi$)

**풀이**:

그래프로 해결:

```
      y
    1 |    ╱╲    
  1/2 |___╱──╲___
      |  ╱    ╲  
  ────┼──────────────── x
    O | π/6 5π/6 2π
   -1 |
```

$y = \sin x$의 그래프가 $y = \frac{1}{2}$보다 위에 있는 구간:

$$\frac{\pi}{6} < x < \frac{5\pi}{6}$$

---

#### ② $\cos x \geq a$ 또는 $\cos x \leq a$

**예제**: $\cos x \leq -\frac{\sqrt{3}}{2}$ ($0 \leq x < 2\pi$)

**풀이**:

```
      y
    1 |╲  ╱╲  ╱
-√3/2 |_╲╱──╲╱_
      |        
  ────┼──────────────── x
    O |5π/6  7π/6  2π
   -1 |
```

$y = \cos x$의 그래프가 $y = -\frac{\sqrt{3}}{2}$보다 아래 또는 같은 구간:

$$\frac{5\pi}{6} \leq x \leq \frac{7\pi}{6}$$

---

#### ③ $\tan x \geq a$ 또는 $\tan x < a$

**예제**: $\tan x \geq 1$ ($0 \leq x < \pi$)

**풀이**:

```
      y
      | |
    1 |╱|___
      |╱|  
  ────┼─┼────── x
    O |π/4 π/2
```

$y = \tan x$의 그래프가 $y = 1$보다 위 또는 같은 구간:

$$\frac{\pi}{4} \leq x < \frac{\pi}{2}$$

만약 $0 \leq x < 2\pi$라면: $\frac{\pi}{4} \leq x < \frac{\pi}{2}$ 또는 $\frac{5\pi}{4} \leq x < \frac{3\pi}{2}$

---

### 📝 삼각부등식 예제

**예제 1**: $2\cos x - 1 > 0$ ($0 \leq x < 2\pi$)

**풀이**:

$$2\cos x > 1$$
$$\cos x > \frac{1}{2}$$

```
      y
    1 |╲  ╱╲  ╱
  1/2 |_╲╱──╲╱_
      |        
  ────┼──────────────── x
    O |π/3    5π/3 2π
```

$$0 \leq x < \frac{\pi}{3} \text{ 또는 } \frac{5\pi}{3} < x < 2\pi$$

---

**예제 2**: $\sin^2 x - \sin x < 0$ ($0 \leq x < 2\pi$)

**풀이**:

$$\sin x(\sin x - 1) < 0$$

경우 분석:
- $\sin x > 0$이고 $\sin x - 1 < 0$ → $0 < \sin x < 1$
- $\sin x < 0$이고 $\sin x - 1 > 0$ → 불가능 ($\sin x \leq 1$)

따라서 $0 < \sin x < 1$:

```
      y
    1 |____╱╲____
      |   ╱  ╲   
  ────┼──────────────── x
    O |   π    2π
```

$$0 < x < \pi$$

(단, $x = 0, \pi, 2\pi$에서 $\sin x = 0$이므로 제외)

## 🔑 핵심 정리

### 삼각함수 그래프

| 함수 | 주기 | 치역 | 대칭 | 영점 |
|------|------|------|------|------|
| $\sin x$ | $2\pi$ | $[-1, 1]$ | 원점대칭 | $n\pi$ |
| $\cos x$ | $2\pi$ | $[-1, 1]$ | $y$축대칭 | $\frac{\pi}{2} + n\pi$ |
| $\tan x$ | $\pi$ | $\mathbb{R}$ | 원점대칭 | $n\pi$ |

### 절댓값 포함 그래프

| 그래프 | 주기 | 치역 |
|--------|------|------|
| $\|\sin x\|$ | $\pi$ | $[0, 1]$ |
| $\|\cos x\|$ | $\pi$ | $[0, 1]$ |
| $\|\tan x\|$ | $\frac{\pi}{2}$ | $[0, \infty)$ |

### 그래프 변형

- **진폭**: $y = A\sin x$ → 진폭 $|A|$
- **주기**: $y = \sin(Bx)$ → 주기 $\frac{2\pi}{|B|}$
- **수평이동**: $y = \sin(x - C)$ → 오른쪽으로 $C$
- **수직이동**: $y = \sin x + D$ → 위로 $D$

### 삼각방정식 풀이

1. 기본각 찾기
2. 해당 사분면 확인
3. 주어진 범위 내 모든 해 구하기

### 삼각부등식 풀이

1. 방정식으로 경계값 찾기
2. 그래프 그리기
3. 부등식 만족 구간 찾기
