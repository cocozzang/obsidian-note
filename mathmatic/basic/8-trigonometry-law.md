# 12. 사인법칙과 코사인법칙

## 기본 학습

### ※ 삼각형의 기본 성질

```
      A
     /|\
    / | \
  c/  |h \b
  /   |   \
 /    |    \
B_____|_____C
    a
```

#### (1) 삼각형의 6요소

- **각**: $\angle A$, $\angle B$, $\angle C$ (또는 $A$, $B$, $C$)
- **변**: $a$, $b$, $c$ (각의 대변)

#### (2) 내각의 합

$$A + B + C = 180° = \pi$$

#### (3) 삼각형의 성립 조건

**가장 긴 변 < 나머지 두 변의 합**

예: $a$가 가장 긴 변이면 $a < b + c$

#### (4) 삼각형의 결정 조건

| 주어진 정보                  | 삼각형 결정           |
| ---------------------------- | --------------------- |
| 세 변 (SSS)                  | ○                     |
| 두 변과 끼인각 (SAS)         | ○                     |
| 두 각과 한 변 (ASA, AAS)     | ○                     |
| 두 변과 끼이지 않은 각 (SSA) | △ (0개, 1개, 2개)     |
| 세 각 (AAA)                  | × (크기만 결정, 비율) |

#### (5) 각의 크기와 변의 관계

$$
\begin{cases}
a^2 + b^2 = c^2 & \Rightarrow \angle C = 90° \text{ (직각)} \\
a^2 + b^2 > c^2 & \Rightarrow \angle C < 90° \text{ (예각)} \\
a^2 + b^2 < c^2 & \Rightarrow \angle C > 90° \text{ (둔각)}
\end{cases}
$$

## 01 사인법칙

### (1) 사인법칙의 공식

삼각형 ABC에서 외접원의 반지름을 $R$이라 하면:

$$\frac{a}{\sin A} = \frac{b}{\sin B} = \frac{c}{\sin C} = 2R$$

```
      A
     /|\
    / | \
  c/  |  \b
  /   |   \
 /    |    \
B_____|_____C
      a
    (R)
```

### (2) 증명

**예각삼각형의 경우**:

```
      A
     /|\
    / |h\
  c/  |  \b
  /   |   \
 /    |    \
B_____|_____C
      a
```

높이를 $h$라 하면:

- $h = c\sin B$
- $h = b\sin C$

따라서:
$$c\sin B = b\sin C$$
$$\frac{b}{\sin B} = \frac{c}{\sin C}$$

마찬가지로 $\frac{a}{\sin A} = \frac{b}{\sin B}$도 증명 가능

외접원 반지름 $R$과의 관계는 원주각의 성질로 증명

### (3) 사인법칙의 변형

① **변의 비 = 사인 비**
$$a : b : c = \sin A : \sin B : \sin C$$

② **개별 비율**
$$a = 2R\sin A, \quad b = 2R\sin B, \quad c = 2R\sin C$$

③ **사인 값**
$$\sin A = \frac{a}{2R}, \quad \sin B = \frac{b}{2R}, \quad \sin C = \frac{c}{2R}$$

### 📝 예제

**예제 1**: 삼각형 ABC에서 $a = 8$, $B = 60°$, $C = 45°$일 때, $b$와 $c$를 구하여라.

**풀이**:

$A = 180° - 60° - 45° = 75°$

사인법칙:
$$\frac{a}{\sin A} = \frac{b}{\sin B} = \frac{c}{\sin C}$$

$$\frac{8}{\sin 75°} = \frac{b}{\sin 60°} = \frac{c}{\sin 45°}$$

$$b = \frac{8\sin 60°}{\sin 75°} = \frac{8 \times \frac{\sqrt{3}}{2}}{\sin 75°} = \frac{4\sqrt{3}}{\sin 75°}$$

$$c = \frac{8\sin 45°}{\sin 75°} = \frac{8 \times \frac{\sqrt{2}}{2}}{\sin 75°} = \frac{4\sqrt{2}}{\sin 75°}$$

---

**예제 2**: 삼각형 ABC의 외접원의 반지름이 5이고, $a = 6$일 때, $\sin A$를 구하여라.

**풀이**:

$$\frac{a}{\sin A} = 2R$$
$$\frac{6}{\sin A} = 2 \times 5 = 10$$
$$\sin A = \frac{6}{10} = \frac{3}{5}$$

---

**예제 3**: 삼각형 ABC에서 $a : b : c = 3 : 5 : 7$일 때, $\cos C$를 구하여라.

**풀이**:

$a = 3k$, $b = 5k$, $c = 7k$로 놓으면

제2 코사인법칙 (아래에서 배움):
$$\cos C = \frac{a^2 + b^2 - c^2}{2ab}$$
$$= \frac{(3k)^2 + (5k)^2 - (7k)^2}{2 \times 3k \times 5k}$$
$$= \frac{9k^2 + 25k^2 - 49k^2}{30k^2}$$
$$= \frac{-15k^2}{30k^2} = -\frac{1}{2}$$

## 02 코사인법칙

### (1) 제1 코사인법칙 (잘 안 씀)

```
      A
     /|\
    / | \
  c/  |  \b
  /   |   \
 /    |    \
B_____|_____C
      a
```

① $a = b\cos C + c\cos B$
② $b = c\cos A + a\cos C$
③ $c = a\cos B + b\cos A$

---

### (2) 제2 코사인법칙 ★★★

**공식**:

① $a^2 = b^2 + c^2 - 2bc\cos A$
② $b^2 = c^2 + a^2 - 2ca\cos B$
③ $c^2 = a^2 + b^2 - 2ab\cos C$

**변형** (각을 구할 때):

$$\cos A = \frac{b^2 + c^2 - a^2}{2bc}$$
$$\cos B = \frac{c^2 + a^2 - b^2}{2ca}$$
$$\cos C = \frac{a^2 + b^2 - c^2}{2ab}$$

### (3) 제2 코사인법칙의 증명

```
      A
     /|\
    / |h\
  c/  |  \b
  /   |   \
 /    |    \
B_____|_____C
    x  a-x
      a
```

$BC$에 $A$에서 수선을 내렸을 때:

- $h^2 + x^2 = c^2$ ... ①
- $h^2 + (a-x)^2 = b^2$ ... ②

②-①:
$$(a-x)^2 - x^2 = b^2 - c^2$$
$$a^2 - 2ax + x^2 - x^2 = b^2 - c^2$$
$$a^2 - 2ax = b^2 - c^2$$

여기서 $x = c\cos B$이므로:
$$a^2 - 2a \cdot c\cos B = b^2 - c^2$$
$$a^2 = b^2 + c^2 - 2bc\cos A$$

### 📝 예제

**예제 1**: 삼각형 ABC에서 $a = 7$, $b = 5$, $c = 3$일 때, $\cos C$를 구하여라.

**풀이**:

$$\cos C = \frac{a^2 + b^2 - c^2}{2ab}$$
$$= \frac{7^2 + 5^2 - 3^2}{2 \times 7 \times 5}$$
$$= \frac{49 + 25 - 9}{70}$$
$$= \frac{65}{70} = \frac{13}{14}$$

---

**예제 2**: 삼각형 ABC에서 $b = 6$, $c = 4$, $A = 60°$일 때, $a$를 구하여라.

**풀이**:

$$a^2 = b^2 + c^2 - 2bc\cos A$$
$$= 6^2 + 4^2 - 2 \times 6 \times 4 \times \cos 60°$$
$$= 36 + 16 - 48 \times \frac{1}{2}$$
$$= 52 - 24 = 28$$
$$a = \sqrt{28} = 2\sqrt{7}$$

---

**예제 3**: 삼각형 ABC에서 $a = 5$, $b = 4$, $c = 6$일 때, 가장 큰 각을 구하여라.

**풀이**:

가장 큰 각은 가장 긴 변의 대각 → $C$ (왜냐하면 $c = 6$이 가장 크므로)

$$\cos C = \frac{a^2 + b^2 - c^2}{2ab}$$
$$= \frac{5^2 + 4^2 - 6^2}{2 \times 5 \times 4}$$
$$= \frac{25 + 16 - 36}{40}$$
$$= \frac{5}{40} = \frac{1}{8}$$

$$C = \cos^{-1}\left(\frac{1}{8}\right) \approx 82.8°$$

## 03 삼각형의 넓이

### (1) 두 변과 끼인각을 알 때 ★★★

$$S = \frac{1}{2}bc\sin A = \frac{1}{2}ca\sin B = \frac{1}{2}ab\sin C$$

```
      A
     /|\
    / |h\
  c/  |  \b
  /   |   \
 /    |    \
B_____|_____C
      a
```

**증명**:
$$h = c\sin B$$
$$S = \frac{1}{2} \times a \times h = \frac{1}{2}ac\sin B$$

### 📝 예제

**예제 1**: 삼각형 ABC에서 $a = 6$, $b = 8$, $C = 30°$일 때, 넓이를 구하여라.

**풀이**:

$$S = \frac{1}{2}ab\sin C = \frac{1}{2} \times 6 \times 8 \times \sin 30°$$
$$= \frac{1}{2} \times 6 \times 8 \times \frac{1}{2} = 12$$

---

### (2) 세 변을 알 때 - 헤론의 공식

$$s = \frac{a + b + c}{2} \text{ (둘레의 반)}$$

$$S = \sqrt{s(s-a)(s-b)(s-c)}$$

### 📝 예제

**예제**: 세 변의 길이가 $a = 5$, $b = 6$, $c = 7$인 삼각형의 넓이를 구하여라.

**풀이**:

$$s = \frac{5 + 6 + 7}{2} = \frac{18}{2} = 9$$

$$S = \sqrt{9(9-5)(9-6)(9-7)}$$
$$= \sqrt{9 \times 4 \times 3 \times 2}$$
$$= \sqrt{216} = \sqrt{36 \times 6} = 6\sqrt{6}$$

---

### (3) 외접원의 반지름을 알 때

$$S = \frac{abc}{4R}$$

**증명**:

사인법칙에서 $\sin A = \frac{a}{2R}$, $\sin B = \frac{b}{2R}$, $\sin C = \frac{c}{2R}$

$$S = \frac{1}{2}bc\sin A = \frac{1}{2}bc \times \frac{a}{2R} = \frac{abc}{4R}$$

---

### (4) 내접원의 반지름을 알 때

$$S = rs$$

여기서 $r$은 내접원의 반지름, $s = \frac{a+b+c}{2}$

```
      A
     /|\
  r/ | \r
  /  o  \
 /   |r  \
B_____|___C
```

**증명**:

삼각형을 내접원의 중심에서 세 부분으로 나누면:

$$S = \frac{1}{2}ar + \frac{1}{2}br + \frac{1}{2}cr$$
$$= \frac{r}{2}(a + b + c) = r \times \frac{a+b+c}{2} = rs$$

---

### 넓이 공식 정리

| 주어진 정보              | 공식                          |
| ------------------------ | ----------------------------- |
| 밑변 $a$, 높이 $h$       | $S = \frac{1}{2}ah$           |
| 두 변 $b, c$, 끼인각 $A$ | $S = \frac{1}{2}bc\sin A$     |
| 세 변 $a, b, c$          | $S = \sqrt{s(s-a)(s-b)(s-c)}$ |
| 외접원 반지름 $R$        | $S = \frac{abc}{4R}$          |
| 내접원 반지름 $r$        | $S = rs$                      |

## 04 사각형의 넓이

### (1) 평행사변형

```
    a
  ┌───┐
 b│  θ│b
  └───┘
    a
```

$$S = ab\sin\theta$$

또는 밑변 × 높이

---

### (2) 일반 사각형

대각선이 $p$, $q$이고, 두 대각선이 이루는 각이 $\theta$일 때:

```
    ╱╲
   ╱  ╲
  ╱ θ  ╲
 ╱      ╲
╱________╲
```

$$S = \frac{1}{2}pq\sin\theta$$

**증명**:

사각형을 대각선으로 나눈 두 삼각형의 넓이 합:

$$S = \frac{1}{2}pq_1\sin\theta + \frac{1}{2}pq_2\sin\theta$$
$$= \frac{1}{2}p(q_1 + q_2)\sin\theta = \frac{1}{2}pq\sin\theta$$

---

### 📝 종합 예제

**예제 1**: 삼각형 ABC에서 $a = 3$, $b = 4$, $c = 5$일 때:
(1) 각 $C$를 구하여라.
(2) 넓이를 구하여라.
(3) 외접원의 반지름을 구하여라.

**풀이**:

(1) $c^2 = a^2 + b^2$인지 확인:
$$5^2 = 3^2 + 4^2$$
$25 = 9 + 16 = 25$ ✓

따라서 $C = 90°$ (직각삼각형)

(2) 넓이:
$$S = \frac{1}{2} \times 3 \times 4 = 6$$

(3) 외접원의 반지름:

직각삼각형의 외접원의 반지름은 빗변의 절반:
$$R = \frac{c}{2} = \frac{5}{2}$$

또는 사인법칙 이용:
$$2R = \frac{c}{\sin C} = \frac{5}{\sin 90°} = \frac{5}{1} = 5$$
$$R = \frac{5}{2}$$

---

**예제 2**: 삼각형 ABC에서 $b = 7$, $c = 8$, $A = 120°$일 때:
(1) $a$를 구하여라.
(2) 넓이를 구하여라.

**풀이**:

(1) 코사인법칙:
$$a^2 = b^2 + c^2 - 2bc\cos A$$
$$= 7^2 + 8^2 - 2 \times 7 \times 8 \times \cos 120°$$
$$= 49 + 64 - 112 \times \left(-\frac{1}{2}\right)$$
$$= 113 + 56 = 169$$
$$a = 13$$

(2) 넓이:
$$S = \frac{1}{2}bc\sin A = \frac{1}{2} \times 7 \times 8 \times \sin 120°$$
$$= \frac{1}{2} \times 56 \times \frac{\sqrt{3}}{2} = 14\sqrt{3}$$

---

**예제 3**: 평행사변형 ABCD에서 두 변의 길이가 6, 8이고 한 내각이 60°일 때, 넓이를 구하여라.

**풀이**:

$$S = ab\sin\theta = 6 \times 8 \times \sin 60°$$
$$= 48 \times \frac{\sqrt{3}}{2} = 24\sqrt{3}$$

---

## 🔑 핵심 정리

### 사인법칙

$$\frac{a}{\sin A} = \frac{b}{\sin B} = \frac{c}{\sin C} = 2R$$

- 변의 비 = 사인 비
- 외접원의 반지름과 관계

### 코사인법칙

$$a^2 = b^2 + c^2 - 2bc\cos A$$

$$\cos A = \frac{b^2 + c^2 - a^2}{2bc}$$

- 세 변으로 각 구하기
- 두 변과 끼인각으로 나머지 변 구하기

### 삼각형의 넓이

| 조건           | 공식                          |
| -------------- | ----------------------------- |
| 두 변 + 끼인각 | $S = \frac{1}{2}ab\sin C$     |
| 세 변          | $S = \sqrt{s(s-a)(s-b)(s-c)}$ |
| 외접원 반지름  | $S = \frac{abc}{4R}$          |
| 내접원 반지름  | $S = rs$                      |

### 각의 크기 판정

- $a^2 + b^2 = c^2$ → 직각
- $a^2 + b^2 > c^2$ → 예각
- $a^2 + b^2 < c^2$ → 둔각
