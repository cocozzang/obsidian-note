# 10. 삼각함수(3) - 그래프 변형

## 01 삼각함수 그래프의 평행이동과 대칭

### <정리> 삼각함수 그래프 변형

#### [1] $y = A\sin(Bx + C) + D$의 그래프

① **진폭** (Amplitude): $|A|$
   - 최댓값과 최솟값의 차이의 절반
   - 최댓값: $|A| + D$
   - 최솟값: $-|A| + D$

② **주기** (Period): $\frac{2\pi}{|B|}$
   - $B > 1$: 주기 짧아짐 (그래프 압축)
   - $0 < B < 1$: 주기 길어짐 (그래프 확장)

③ **위상** (Phase shift): $-\frac{C}{B}$
   - $x$축 방향 평행이동
   - $C > 0$: 왼쪽으로 $\frac{C}{B}$
   - $C < 0$: 오른쪽으로 $\frac{|C|}{B}$

④ **수직이동** (Vertical shift): $D$
   - $y$축 방향 평행이동
   - 중심선: $y = D$

---

### 📝 예제

**예제 1**: $y = 3\sin\left(2x - \frac{\pi}{3}\right) + 1$의 그래프를 분석하여라.

**풀이**:

$$y = 3\sin\left(2\left(x - \frac{\pi}{6}\right)\right) + 1$$

- **진폭**: $|A| = 3$
- **주기**: $\frac{2\pi}{2} = \pi$
- **위상**: $x$축으로 $\frac{\pi}{6}$ (오른쪽)
- **수직이동**: $y$축으로 $1$ (위로)
- **최댓값**: $3 + 1 = 4$
- **최솟값**: $-3 + 1 = -2$
- **중심선**: $y = 1$

```
      y
    4 |    ╱╲    ╱
    1 |___╱──╲__╱___
   -2 |  ╱    ╲╱
  ────┼──────────────── x
    O | π/6   7π/6
```

---

**예제 2**: $y = -2\cos\left(\frac{1}{2}x + \pi\right) - 1$의 그래프를 분석하여라.

**풀이**:

$$y = -2\cos\left(\frac{1}{2}\left(x + 2\pi\right)\right) - 1$$

- **진폭**: $|-2| = 2$
- **주기**: $\frac{2\pi}{1/2} = 4\pi$
- **위상**: $x$축으로 $2\pi$ (왼쪽)
- **수직이동**: $y$축으로 $-1$ (아래로)
- **최댓값**: $-(-2) + (-1) = 1$ (음수이므로 위아래 반전)
- **최솟값**: $-2 + (-1) = -3$
- **중심선**: $y = -1$

---

**예제 3**: $y = \frac{1}{2}\sin(3x) + 2$의 진폭, 주기, 최댓값, 최솟값을 구하여라.

**풀이**:

- **진폭**: $\frac{1}{2}$
- **주기**: $\frac{2\pi}{3}$
- **최댓값**: $\frac{1}{2} + 2 = \frac{5}{2}$
- **최솟값**: $-\frac{1}{2} + 2 = \frac{3}{2}$
- **중심선**: $y = 2$

---

## 02 삼각함수 그래프 변형 단계별 정리

### (1) $y = 2\sin x$

**변환**: 진폭 2배

```
      y
    2 |    ╱╲    
    1 |   ╱  ╲   
      |  ╱    ╲  
  ────┼──────────────── x
    O | π/2  π  3π/2
   -1 |     ╲  ╱
   -2 |      ╲╱
```

- 주기: $2\pi$ (변화 없음)
- 진폭: $2$

---

### (2) $y = \sin 2x$

**변환**: 주기 절반 ($\pi$)

```
      y
    1 |  ╱╲╱╲╱╲╱╲
      | ╱        ╲
  ────┼──────────────── x
    O | π/2  π  3π/2
   -1 |
```

- 주기: $\pi$
- 한 주기 안에 파동 2개

---

### (3) $y = \sin\left(x - \frac{\pi}{2}\right)$

**변환**: $x$축으로 $\frac{\pi}{2}$ 오른쪽 이동

```
      y
    1 |      ╱╲    ╱╲
      |     ╱  ╲  ╱  
      |    ╱    ╲╱   
  ────┼──────────────────── x
    O | π/2  π  3π/2 2π
   -1 |         ╲  ╱
```

$$y = \sin\left(x - \frac{\pi}{2}\right) = -\cos x$$

---

### (4) $y = \sin x + 3$

**변환**: $y$축으로 $3$ 위로 이동

```
      y
    4 |    ╱╲    
    3 |___╱──╲___
    2 |  ╱    ╲  
  ────┼──────────────── x
    O | π/2  π  3π/2
```

- 중심선: $y = 3$
- 최댓값: $4$, 최솟값: $2$

---

### (5) $y = -\sin x$

**변환**: $x$축 대칭 (상하 반전)

```
      y
    1 |
      |  ╲╱    ╲╱
      | ╱  ╲  ╱  ╲
  ────┼──────────────── x
      | π/2  π  3π/2
   -1 |╱    ╲╱    ╲
```

- $A < 0$이면 그래프 반전

---

### (6) $y = \sin(-x)$

**변환**: $y$축 대칭

```
      y
    1 |╲    ╱╲    ╱
      | ╲  ╱  ╲  ╱
      |  ╲╱    ╲╱
  ────┼──────────────── x
      | π/2  π  3π/2
   -1 |
```

$$y = \sin(-x) = -\sin x$$

- $\sin$은 기함수이므로 결과적으로 $x$축 대칭

## 03 탄젠트 함수의 그래프 변형

### (1) $y = \tan x$ (기본형)

```
      y
    3 |  |     |     |
    2 |  |   ╱ |   ╱ |
    1 | ╱|  ╱  |  ╱  |
  ────┼──┼─────┼─────┼──── x
   -π/2| O  π/2  π  3π/2
   -1 | |     |     |
```

- 주기: $\pi$
- 점근선: $x = \frac{\pi}{2} + n\pi$

---

### (2) $y = \tan 2x$

**변환**: 주기 절반 ($\frac{\pi}{2}$)

```
      y
      | | | | |
    2 |╱|╱|╱|╱|
    1 |╱|╱|╱|╱|
  ────┼─┼─┼─┼─┼──── x
  -π/4|O π/4 π/2
   -1 | | | | |
```

- 주기: $\frac{\pi}{2}$
- 점근선: $x = \frac{\pi}{4} + \frac{n\pi}{2}$

---

### (3) $y = \tan\left(x - \frac{\pi}{4}\right)$

**변환**: $x$축으로 $\frac{\pi}{4}$ 오른쪽 이동

```
      y
      |   |     |
    1 |  ╱|   ╱ |
  ────┼───┼─────┼──── x
    O | π/4  3π/4
```

- 점근선도 함께 이동: $x = \frac{3\pi}{4}, \frac{7\pi}{4}, ...$

## 04 복합 그래프 변형

### 📝 예제

**예제 1**: $y = 2\sin\left(3x + \frac{\pi}{2}\right) - 1$을 그래프로 나타내고 주요 정보를 구하여라.

**풀이**:

$$y = 2\sin\left(3\left(x + \frac{\pi}{6}\right)\right) - 1$$

**분석**:
- 진폭: $2$
- 주기: $\frac{2\pi}{3}$
- 위상: 왼쪽으로 $\frac{\pi}{6}$
- 수직이동: 아래로 $1$
- 최댓값: $2 + (-1) = 1$
- 최솟값: $-2 + (-1) = -3$

**그래프 그리기 순서**:
1. $y = \sin x$ (기본)
2. $y = 2\sin x$ (진폭 2)
3. $y = 2\sin 3x$ (주기 $\frac{2\pi}{3}$)
4. $y = 2\sin\left(3x + \frac{\pi}{2}\right)$ (왼쪽 $\frac{\pi}{6}$)
5. $y = 2\sin\left(3x + \frac{\pi}{2}\right) - 1$ (아래로 $1$)

```
      y
    1 |╲  ╱╲  ╱╲  ╱
   -1 |_╲╱──╲╱──╲╱_
   -3 |  ╲╱    ╲╱
  ────┼──────────────── x
  -π/6|   π/3  5π/6
```

---

**예제 2**: $y = -3\cos\left(\frac{1}{2}x - \frac{\pi}{4}\right) + 2$의 그래프 정보를 구하여라.

**풀이**:

$$y = -3\cos\left(\frac{1}{2}\left(x - \frac{\pi}{2}\right)\right) + 2$$

- 진폭: $3$
- 주기: $\frac{2\pi}{1/2} = 4\pi$
- 위상: 오른쪽으로 $\frac{\pi}{2}$
- 수직이동: 위로 $2$
- **부호**: 음수이므로 상하 반전
- 최댓값: $3 + 2 = 5$ (반전으로 인해)
- 최솟값: $-3 + 2 = -1$

---

**예제 3**: $y = \frac{1}{2}\tan\left(2x + \frac{\pi}{3}\right)$의 주기와 점근선을 구하여라.

**풀이**:

$$y = \frac{1}{2}\tan\left(2\left(x + \frac{\pi}{6}\right)\right)$$

- 주기: $\frac{\pi}{2}$
- 위상: 왼쪽으로 $\frac{\pi}{6}$

**점근선**:
$2x + \frac{\pi}{3} = \frac{\pi}{2} + n\pi$

$$2x = \frac{\pi}{2} - \frac{\pi}{3} + n\pi = \frac{\pi}{6} + n\pi$$

$$x = \frac{\pi}{12} + \frac{n\pi}{2}$$

점근선: $x = \frac{\pi}{12}, \frac{7\pi}{12}, \frac{13\pi}{12}, ...$

## 05 특수한 그래프 변형

### (1) $y = \sin|x|$

**변환**: $x < 0$ 부분을 $y$축 대칭

```
      y
    1 |  ╱╲  ╱╲  
      | ╱  ╲╱  ╲ 
  ────┼──────────── x
      | π  O  π
   -1 |
```

- $x \geq 0$: $y = \sin x$
- $x < 0$: $y = \sin(-x) = -\sin x$를 반전 → $y = \sin(-x)$

---

### (2) $y = |\sin x|$ (복습)

**변환**: $y < 0$ 부분을 $x$축 대칭

```
      y
    1 |╲╱╲╱╲╱╲╱
      |        
  ────┼──────────── x
    O | π  2π  3π
```

- 주기: $\pi$ (절반)

---

### (3) $y = \sin x + \cos x$

**변환**: 삼각함수의 합성

**공식**:
$$\sin x + \cos x = \sqrt{2}\sin\left(x + \frac{\pi}{4}\right)$$

**증명**:
$$a\sin x + b\cos x = \sqrt{a^2 + b^2}\sin(x + \alpha)$$

여기서 $\cos\alpha = \frac{a}{\sqrt{a^2 + b^2}}$, $\sin\alpha = \frac{b}{\sqrt{a^2 + b^2}}$

$a = 1, b = 1$인 경우:
- $\sqrt{1^2 + 1^2} = \sqrt{2}$
- $\cos\alpha = \frac{1}{\sqrt{2}}$, $\sin\alpha = \frac{1}{\sqrt{2}}$ → $\alpha = \frac{\pi}{4}$

따라서:
$$\sin x + \cos x = \sqrt{2}\sin\left(x + \frac{\pi}{4}\right)$$

**그래프**:
```
      y
   √2 |    ╱╲    
      |   ╱  ╲   
  ────┼──────────────── x
    O | π/4  5π/4
  -√2 |     ╲╱
```

- 진폭: $\sqrt{2}$
- 주기: $2\pi$
- 위상: 왼쪽으로 $\frac{\pi}{4}$

---

### 📝 예제

**예제**: $y = \sqrt{3}\sin x - \cos x$를 $y = A\sin(x + \alpha)$ 꼴로 나타내어라.

**풀이**:

$a = \sqrt{3}$, $b = -1$

$$A = \sqrt{(\sqrt{3})^2 + (-1)^2} = \sqrt{3 + 1} = 2$$

$$\cos\alpha = \frac{\sqrt{3}}{2}, \quad \sin\alpha = \frac{-1}{2}$$

이를 만족하는 $\alpha$: $\alpha = -\frac{\pi}{6}$ (또는 $\frac{11\pi}{6}$)

$$y = 2\sin\left(x - \frac{\pi}{6}\right)$$

## 🔑 핵심 정리

### 삼각함수 그래프 변형 공식

$$y = A\sin(Bx + C) + D$$

| 요소 | 의미 | 계산 |
|------|------|------|
| $A$ | 진폭 | $\|A\|$ |
| $B$ | 주기 결정 | 주기 = $\frac{2\pi}{\|B\|}$ |
| $C$ | 위상 이동 | $x$축으로 $-\frac{C}{B}$ |
| $D$ | 수직 이동 | 중심선 $y = D$ |

### 최댓값과 최솟값

- $A > 0$: 최댓값 $= A + D$, 최솟값 $= -A + D$
- $A < 0$: 최댓값 $= -A + D$, 최솟값 $= A + D$ (반전)

### 주기 변화

| 함수 | 주기 |
|------|------|
| $\sin x$ | $2\pi$ |
| $\sin(Bx)$ | $\frac{2\pi}{\|B\|}$ |
| $\tan x$ | $\pi$ |
| $\tan(Bx)$ | $\frac{\pi}{\|B\|}$ |
| $\|\sin x\|$ | $\pi$ |

### 삼각함수의 합성

$$a\sin x + b\cos x = \sqrt{a^2 + b^2}\sin(x + \alpha)$$

여기서 $\cos\alpha = \frac{a}{\sqrt{a^2 + b^2}}$, $\sin\alpha = \frac{b}{\sqrt{a^2 + b^2}}$
